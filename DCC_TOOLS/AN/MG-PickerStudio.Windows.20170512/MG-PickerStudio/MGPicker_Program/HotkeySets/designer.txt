x : Text Tool

m : Mirror Tool
i : Copy / Paste Property Tool
t : Global Transform Tool
r : Local Transform Tool
p : Polygon Tool
b : Bezier Tool

enter : Confirm Edit, Hold Alt To Confirm Edit To Mirrored Items As Well
up : Move Items Up 1, Hold Alt To Move Mirrored Items As Well
dn : Move Items Down 1, Hold Alt To Move Mirrored Items As Well
shift up : Move Items Up 5, Hold Alt To Move Mirrored Items As Well
shift dn : Move Items Down 5, Hold Alt To Move Mirrored Items As Well
lt : Move Items Left 1, Hold Alt To Move Mirrored Items As Well
rt : Move Items Right 1, Hold Alt To Move Mirrored Items As Well
shift lt : Move Items Left 5, Hold Alt To Move Mirrored Items As Well
shift rt : Move Items Right 5, Hold Alt To Move Mirrored Items As Well

ctrl alt s : Save Picker To Other Directory

ctrl alt shift e : Export Picker File Path To Node

ctrl dn : Send Selected Items To Bottom Layer

shift m : Apply Minimum Info To Mirrored

ctrl shift h : Align Panel via Horizontal Center

ctrl shift alt v : Align Vertical Center
ctrl shift alt r : Align Right
ctrl shift alt b : Align Bottom

ctrl alt v : Distribute Vertically

ctrl u : Adjust Button Color

ctrl l : Label Selected Buttons Automatically
backspace : Clear Label
shift backspace : Delete Selected Items, Or Delete Current Point During Polygon / Bezier Edit.
