▪Quick picker design workflow:

- Choice #1: Create picker from a existing picker template:

-3 ways to create picker from template:

1. ALT+LMB double clicking on the free space of tab area, or the empty space of the picker area if no picker been loaded.

2. Use the "CTRL + ALT + N" hotkey.

3. Select the menu  Picker / New Picker From Template..

-Select a picker template, input a picker name and hit "OK" button, you will create a new picker from the template.

 

- Choice #2: Create from scratch but utilizing several on-click features:

1.Create new picker file.
Use hotkey "CTRL + N", or CTRL + LMB double clicking on the free space of tab area, or seelct menu Picker / New Picker.., to create a new picker file.
There will be a panel by default in the new picker file.

2. Snapshot picker panel background image.
Make sure you are in the edit tool:
LMB  click on the top right triangle of the panel title area, Select "1. Prepare Snapshot".
This sets up the resolution gate of the 3d view, also if there is children items in the panel, a snapshot of panel content will be made and set as imageplane of the 3d view, serves as a reference.
Tweak the 3d view, prepare to take the background snaphot image for the picker panel , usually you wanna use front camera to do this.
Select "2. Snaphot" to take the snapshot and set the background image as picker panel.

3.1-Click create picker select buttons.
Select the controls you wanna created as picker select button.
LMB  click on the top right triangle of the panel title area, Select "1. Prepare 3D View", usually you wanna use front camera to do this.
Tweak the 3d view, so that all the controls are visible in the 3d view.
Select "2. Add SelectButtons Via 3DView" to make all the select-buttons.
Move items to make sure they are not overlapping with each other.
Select the select-button you wanna label, press "CTRL + L" to autolabel them.

4.Link the select-buttons and setup the mirror relationship.
Use tool to link the select-button together, so that in future animator could double click the parent select-button to select all the child select-buttons members.
Use tool click on each pair of select-buttons, mark them as mirred buttons.
You could also frame select the buttons at one side, and right click on them,select "Auto-Assign Mirror Relationship" to do the job.
There are many advantages to always assign the mirrorship: In design phase, move them reflectively, in animator mode, mirror selection, move slider handle reflectively, etc.

5.Add Command Buttons.
In edit tool or command-button tool , right click on the view, select "Add Command Button" and select a command button preset to create command button.
Usually you add the select all button, key all button, etc.

6.1-Click add Pose Command Buttons.

You could add pose buttons, which are just command button with images.

In edit tool or command-button tool , right click on the view, select "Add Pose Command Button" to add a pose buton without thumbnail image.

Or select "Pose Command Button With Icon", prepare the 3d view, and snapshot and create a pose button with icon.

7.1-Click create picker sliders for your facial panel.
You probably need to create a new picker panel for the facial sliders. Use hotky "CTRL + SHIFT + N" to add a new panel.
Select all the controls in your nurbs-curve based facial panel,
LMB  click on the top right triangle of the panel title area, Select "1. Prepare 3D View", and tweak camera view so that all curve sliders are visible in the view.
Select "2. Add Sliders Via 3DView" to make all the sliders.
Move the overlapping sliders to layout all the sliders, also "CTRL + L" to auto-label them.
Repeat the step 4 on these sliders, to link them, and mark the mirror relationships.

8.Edit, tweak, test, and done!
If you wanna test the picker, you switch to the preview tool , to test all the button interactivity.
You could press "F8" to switch to real animator mode, but it is risky because in that mode the unsaved edit is not savable and will not be warned when you close the picker.
