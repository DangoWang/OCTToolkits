▪Item shapes:

A picker item, which could be select-button, command-button, slider, attribute-button, text or panel, all of them contain a shape of their own.

The shape is in charge of drawing the picker item, it decide how the item will be looking like.

There are many types of item shape:

Rectangle, Rounded rectange,Ellipse, Circle,Triangle, Diamond, Trapezoid,Pentagon, Hexagon,Polygon, Bezier,Image,Widget Control, Attribute Button, Rounded Attribute Button,Slider.

Some picker item only support a certain shape type, such as Widget Control, Attribute Button, Rounded Attribute Button shape types could only be used by attribute-button,

slider shape could only be used by slider item.

You could switch to other shape that the picker supports in top of attribute editor.

Different shape types has different properties, which could also be set in attribute editor.

To use the polygon tool to draw/edit polgyon, you should select a picker contain a polygon shape. If not, switch its shape type to polygon.

To use the bezier tool to draw/edit bezier curve, you should select a picker contain a bezier shape. If not, switch its shape type to bezier.

To rotate / flip the bezier curve or polygon shape, select the picker item, and  go to Edit Menu / Transform Bezier/Polygon Item..
