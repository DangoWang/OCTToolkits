▪Tips for using the MG-Picker Studio in animator mode:

Use "Ctrl+F" hotkey and "Escape" hotkey to turn on / off the attribute searching widget.

In animator mode, MG-Picker Studio only occupy very small set of hotkeys, you could use almost each maya hotkey. In designer mode, only picker hotkeys could be accessed.

When you do the frame selection or click on a select-buton or a slider handle, thing will be selected or deselected in maya scene, meanwhile, it will automatically refocus on maya 3d view, so maya's hotkey works, say, press "S" to set key.

A post-action could be set for a select-button during the design phase, so that after you click and release on it, MG-Picker studio will switch your maya to move/rotate/scale tools automatically.

CTRL + Tab hotkey to switch to next picker. CTRL + SHIFT + Tab hotkey to switch to previous picker.

In the pop up picker switcher, MMB click on a picker thumbnail to close that picker.

Attribute link will be invisible in animator mode, while parent link will be visible with a configable opacity,

they are not totally invisible because animator need to know what child select-buttons will also be used for selecting when he doubles clicking on the parent select-button.

You could set the parent link opacity in MG-Picker Studio preference window.
