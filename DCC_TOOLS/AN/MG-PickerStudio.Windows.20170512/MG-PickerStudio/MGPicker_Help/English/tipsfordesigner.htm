▪Tips for using the MG-Picker Studio in designer mode:

These is tool-tip for each menu items, UI buttons, property setting widgets. Read them to know what its use.

When you snapshot image for picker, you can apply a automatic image filter to the image once after it is generated.

Select "Misc Menu / Image Style" to bring up the image processing dialog, where you could edit and add your own style of image processing.

Every picker file / node could attach some information of picker designer.

To do this, select "Picker Menu / Set Picker Meta Info..", the dialog that pops up enable you to add your name, email, website and message into the picker file / node.

Convert normal shapes to polygon, so that you could use polygon tool to edit and draw on it:

Select the picker item, select the "Edit Menu / Convert Items To Polygon Shape".

This feature not always works, depend on the current shape type adopted by the select item, for now some shape type can't be converted to polygon, such as ellipse, circle, etc.

If the picker item is moved beyond a panel content area:

Select "Edit Menu / Move Back Out-Ranged Items". The out bounded items will be moved back to show themselves.

This will apply to all the picker items within current picker file/node.

You could also LMB click on the top right triangle of a picker panel's title area, select "Move Back Out-Ranged Buttons", this will only apply to current panel.

If the picker item has been hidden due to attribute button attribute change:

Sometimes you connect picker items to attribute button using attribute link.

You could change the value of attribute button to make them invisible, and set back the target value that the attribute link was created at to make them visible again.

But for somehow this doesn't work anymore, eg. you change the attribute button's attribute type, refer it to other maya attribute, etc.

You could always select "Edit menu / Show All Hidden Items" to show them by force, then you could delete the attribute links and relink them at new target values.
