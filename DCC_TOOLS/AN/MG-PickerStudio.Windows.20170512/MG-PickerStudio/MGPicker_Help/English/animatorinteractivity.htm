▪Animator interactivity:

MG-Picker provide powerful and very rich ways for animator to interactive with pickers.

Maker sure you are in animator interactive mode, which mean this button is being selected:

- Selecting: The same selecting rule as maya.

LMB click on select-button / Frame selecting: Replace selection.

CTRL + LMB click on select-button / Frame selecting: Deselect.

SHIFT + LMB click on select-button / Frame selecting: Toggle Selection

Holding ALT while you doing selecting, the same select rule will applied to the select-buttons' mirror buttons,

which means hold ALT frame select a select-button, its mirror button's member will also be selected / deselected / toggle selection.

LMB + Double clicking on a select-button, will select the clicked button's members, as well as all the child select-buttons' members.

To utilize this feature,  the select-buttons should be linked with parent links.

CTRL / SHIFT + RMB click on select-button: will popup the button's member view to perform more in-depth selection.  Same select rules applied to the member view.

Click the tearoff button of the member view, you just made the popup member view inito a normal MG-Select Helper window.

 

- Command Button: Just LMB click and release on a command button to execute its codes.

- Pose command button： LMB click on it to apply 100% the pose, CTRL + LMB clicking on it to apply 20% for each click, SHIFT + LMB clicking on it to apply 50% for each click.

 

- Attribute Button: LMB click on value area of attribute button, will trigger its value change.

Click on label area of attribute button will simple select that attribute button for MMB attr-nudging, unless the attribute button's fill color is transparent enough.

 - Each time an attribute button is selected:

The current tool of maya will automatically be switched to attr-nudger tool comes with MG-Picker Studio,

hence you could MMB drag in the 3d viewport, drag in horizontal direction to nudge the attribute controlled by the attribute button(s).

 

- Slider : Same CTRL / SHIFT / Alt / Double Clicking select rule apply to slider handles regarding of the maya nodes selection.

- When moving a slider handle in picker view:

SHIFT + LMB draging: Contraint the slider handle moving direction in horizontal / vertical. The only make sense for a 2D slider, for a 1D slider, only 1 direction available.

Alt + LMB draging: Moving the slider's mirror slider handle as well. Which means, for instance, you could tweak the left / right mouse control in reverse way regarding their values.

- To quickly reset the attribute value of a slider:

Select all the sliders you want to reset their value, double clicking on the background one of the selected slider.

Be careful not double clicking the slider handle, which will select the clicked slider's maya members as well as the child slider handles'.

If the attribute that the slider stands for doesn't exist in your scene, the slider handle wont move if you double click its slot.

If you directly LMB double click on a unselected slider's background, will only reset that clicked slider's attribute values.

- Each time a slider handle is selected:

The current tool of maya will automatically be switched to attr-nudger tool comes with MG-Picker Studio,

hence you could MMB drag in the 3d viewport, drag in horizontal direction will nudge the H-attribute while drag in vertical direction nudges the V-attribute.

The attr-nudger tool doesn't support ALT to nudge the mirrored slider attribute, since alt key in maya is for viewport navigation,

but it does support SHIFT key while being MMB dragged, which constrains the dragging direction to horizontal / vertical, and the CTRL key to nudge the attributes in smaller steps.

 

- Left / Right Clicking Menu of picker item: Each picker item could be attached a left / right clicking menu with it, even the picker panel.

This is done in picker designer phase.

For each item that contains a right clicking menu, there will be a small solid triangle in the shape. Just right click on the item to pop the menu.

For each item that contains a left clicking menu, there will be a small angle bracket sign in the shape. Just left click on the item to pop the menu.

The menu could be a marking menu.

You could right click on a picker panel's content area to use its right clicking menu, for left clicking menu, the only choice is left clicking the panel's title area.
