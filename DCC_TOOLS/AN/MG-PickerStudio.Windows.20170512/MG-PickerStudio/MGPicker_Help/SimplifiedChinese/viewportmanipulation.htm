▪Picker视图操作:

-MG-Picker视图操作采用跟Maya的3D视图相似的操作规则:

鼠标中键: 平移视图，不管有没有按ALT键。

ALT + 鼠标右键: 放大/缩小视图。

CTRL + ALT + 鼠标左键: 框选的方式放大局部/重置到默认大小。

F 键: 放大选中的Picker对象，如果没选择任何Picker对象则是缩小视图以显示所有的Picker对象。

空格键：如果当前您处于多重Picker视图布局，快速按空格键并松开，会放大鼠标下的那个视图区域而隐藏其它区域，再按空格键来恢复。

 

-在上同操作规则之外，还有其它的额外的规则:

ALT + 鼠标右键双击: 重置到默认大小。

鼠标中键滑轮: 取决于当前Picker内容的长宽比，可能在水平/垂直方向上平移视图。
 在鼠标中键滑轮时，如果加按ALT 键则会以相反的方向平移视图。

Ctrl + 鼠标中键滑轮: 放大/缩小视图。

0~9 键: 自动缩放及平移视图，来显示某一个Picker面板。每个Picker面板有一个0~9间的面板索引对应这0~9的快捷键。
您可以使用 ~ 键来代替0键.

 

-多重Picker视图:

 -通常一个Picker文件代表一个资产，如角色，道具或场景。要同时显示多个资产的Picker：

鼠标左键点击Picker视图区右上角的布局切换按钮，并选择一个布局类型。

除第一个是单一视图外，其它都是多重视图，最多同时显示四个。在多重视图情况下，按空格键来快速放大并隐藏一个视图区域。

你可以用鼠标左键拖拽Picker标签，把相应的Picker视图拖到新加入的这些视图区。或直接用鼠标左键激活这些新区域，并加载Picker到这个区域。
