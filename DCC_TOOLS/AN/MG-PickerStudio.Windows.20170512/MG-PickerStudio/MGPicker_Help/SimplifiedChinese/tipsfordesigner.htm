▪MG-Picker工作室在设计师模式下的小技巧:

每一个菜单项，界面按钮，属性控件都有鼠标经过提示，可以阅读这些提示来了解它的用途。

当您为picker截图生成图片时，比如为面板生成背景图片，你可以应用一个自动化的图片处理预设。这个预设在截图产生后自动被应用，像photoshop的滤镜。

如果要编辑这些图片处理预设，选择 "杂项菜单 / 图片处理样式" 来打开图片处理对话框，在那里你可以生成，编辑图片处理预设。

每个picker文件或节点都可以加上设计师的信息。

选择“Picker菜单 / 设置picker文件属性..”，在弹出的对话框里您可以输入您的名字，网站，邮箱及想要给picker使用者的一段话。

可以将一般形状转换为多边形形状，以便您可以使用多边形工具来编辑绘制它。

选择一些非多边形状态的对象，选择“编辑菜单 / 将对象转换为多边形形状”。

注意这个功能不是对所有形状都有效，有些形状如椭圆，贝塞尔曲线则使用这个功能无效。

如果一个picker对象被移到面板内容区域以外的区域而变成不可见:

选择“编辑菜单  / 移回超出范围的按钮” 来将这些按钮给移回来。这将应用于整个picker文件或节点，

您也可以点击picker面板右上方的三角，在菜单里也有类似功能，不过它将只移回当前picker面板里超出的对象。

如果picker对象因为属性按钮的数值改变而隐藏：

有时你会将一些picker对象用属性连接连给一个属性按钮来控制它们的可见性。

你可以通过属性按钮数值改变来使这些对象不可见，也可以通地设置回连接时采用的目标属性值来使它们重新可见。

不过有些情况下无法将它们重新变回可见，比如你改变了属性按钮的属性类型，或将它指向另外一个完全不同的maya属性，等等。

这里你可以选择“编辑菜单 / 显示所有被隐藏对象” 来使这些被隐藏对象重新可见，这样你可以删除这些属性连接，重新连过。
