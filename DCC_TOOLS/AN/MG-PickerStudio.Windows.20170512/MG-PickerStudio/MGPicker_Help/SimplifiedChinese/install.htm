▪ 第一次安装MG-Picker工作室：

1.访问 http://twincodes.com 或 http://mgland.com 下载最新的MG-Picker工作室程序并解压到您的本地硬盘。

2.不要变更里面的任何文件/文件夹，用鼠标左键拖拽"MGPicker_installer.mel"到运行中的Maya的3D视图并松开。

3.根据安装向导安装，基本上只有一步步点下一步即可。在界面语言那一步，切换到SimplifiedChinese，以便使用中文界面。

4.安装模式如果选"拷贝到Maya的用户脚本目录再调用"，则所有的程序将被安装到当前Maya的脚本文件夹/MG-PickerStudio文件夹。
如果是选"不拷贝，直接原地调用程序"，则只会生成指向安装程序的工具架按钮，而不做任何拷贝的动作。
选用第二种模式比较方便，因为程序有更新时只要下载并覆盖那个统一目录就好。不过请确保这个程序目录不包含中文路径。

5.在Maya的用户程序文件夹下面，MG_PickerData文件夹被创建，以便将来存储用户的本地picker数据，这些数据将被本地的所有Maya版本所共用。

 

▪ 安装MG-Picker工作室到其它Maya版本：

1.找到已安装MG-Picker工作室的Maya版本所对应的用户脚本文件夹。

2.找到其下的MG-PickerStudio/MGPicker_Program/Installer/MGPicker_installer.mel这个安装MEL。

3.用鼠标左键拖拽"MGPicker_installer.mel"到运行中的其中要安装到的Maya版本的3D视图并松开。

4.据安装向导安装，基本上只有一步步点下一步即可。在界面语言那一步，切换到SimplifiedChinese，以便使用中文界面。

5.安装模式如果选"拷贝到Maya的用户脚本目录再调用"，则所有的程序将被安装到当前Maya的脚本文件夹/MG-PickerStudio文件夹。
如果是选"不拷贝，直接原地调用程序"，则只会生成指向安装程序的工具架按钮，而不做任何拷贝的动作。
选用第二种模式比较方便，因为程序有更新时只要下载并覆盖那个统一目录就好。不过请确保这个程序目录不包含中文路径。
 

▪ 卸载MG-Picker工作室：

必须切到设计师模式下，找到杂项菜单，并选择"卸载MG-Picker工作室.."，这样将弹出卸载窗口，选择要卸载的选项，直接点"卸载 !!"按钮即可卸载。

注意，如果之前的安装模式是选的"不拷贝，直接原地调用程序"，则作为程序源的目录不会被卸载（因为有可能其它maya还需要）。

如果你要卸载这模式的MG-Picker工作室，则先用卸载程序卸载，清除掉那些工具架按钮，自动加载机制等，然后再手动删除那个程序源目录。
