▪MG-Picker工作室动画师模式小提示:

使用 "Ctrl+F" 及 "Esc" 快捷键来在显示/隐藏属性搜索界面。

你可以直接在动画师模式下使用maya快捷键。在设计师模式下，则只能使用picker的快捷键，无法使用maya的。

在你使用选择按钮或滑条滑杠影响了maya选择后，工具会自动把焦点切回到maya的3D视图，这样所有maya快捷键仍然可用，比如用来key帧的S键。

在设计阶段可以为每个选择按钮指定一个选择后行为，这些行为即是在按钮被选择后，自动切换到位移/旋转/缩放等工具。

使用CTRL + Tab快捷键来切到下一个Picker视图，CTRL + SHIFT + Tab快捷键来跳到上一个Picker视图。

在切换时弹出的窗口里，也可以中键点击一个picker缩略图来关闭那个picker。

属性连接在动画师模式下不可见，不过父子连接则以一个配置的透明度呈现在这个模式下。

父子连接没有被设定为完全不可见是因为动画师可以用它来判断双击父按钮则哪些个子按钮同时被选择。

您可以在配置对话框里设置这个透明度或完全不可见。
