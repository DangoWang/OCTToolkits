▪基本上，你将使用编辑工具，变形工具，复制粘贴属性工具来进行移动，编辑操作。

 

▪变形所选对象

你可以用编辑工具来移动picker对象，在属性编辑器里设置它们的大小。

不过除此外，你可以使用"CTRL + T"快捷键来切到变形工具来变形事先选择的多个按钮以便整体位移，缩放它们。

变形后，用回车键确认变形，或按Esc键来取消变形。

如果按回车键里加上Alt键，则同样对他们所镜象的对象应用镜象后变形效果。

 

▪变形贝塞尔曲线及多边形形状：

你可以旋转或翻转贝塞尔曲线或多边形形状，选择相应的Picker对象，然后点选编辑菜单 / 变形多边形/贝塞尔曲线形状..

 

▪复制对象

选择要复制的对象，按 "CTRL + D" 来复制对象，复制出来的新对象将放在原对象下方。

按"CTRL + SHIFT + D" 来复制并将复制器置于原对象右方。

如果加上alt键，则复制器放在上面规则的相反方向。
 

▪复制/粘贴对象

CTRL + C / CTRL + V 对于picker复制粘贴一样有效。也可以在这个picker复制，另一个picker文件里粘贴。

在复制时，如果所选的要复制对象包含picker面板，则只有这些picker机板被复制，其它类型的对象被忽略。
 

▪复制/粘贴对象属性

切到属性复制粘贴工具 ， 不能可以使用快捷键“I”键。

要复制属性，右键点击要复制属性的对象，在弹出菜单里选择一个复制模式。

然后左键点击其它对象粘贴这些属性。
 

▪对齐/ 均等分布对象

选择要对齐或分布的对象，点击工具栏上的对齐或分布按钮，或编辑菜单里的相应菜单来对齐，均等分布它们。
 

▪缩放对象到合适大小

选择要缩放的对象，按 "CTRL + R" 快捷键来将它们缩放各自的合适大小。在编辑菜单里有同样功能。
 

▪调节对象的填充颜色

你可以在属性编辑器里调整对象的填充色。

不过如果你要以photoshop那样的调色方式一下子调整多个不同颜色的按钮：

选择要调整的所有对象，按 "CTRL + U"打开颜色调节对话框。

它就类似于photoshop的色相和饱和度对话框。
 

▪ 对镜象对象应用属性

使用ALT键。

比如要同时设置镜象标签边距，在原对象设置标签边距，按ALT + 回车键确认，则原对象的镜象对象标签边距也会同样设置，并且把镜象关系考虑进去。
 

▪成为子对象/ 脱离父子对象关系

所有的picker对象类型，包括picker面板本身，都可以成为其它picker面板的子对象。

默认情况下，如果您使用交互创建工具在picker面板上方创建一个picker对象，则这个新对象自动成为该面板的子对象。

不过如果事后想要成为某面板的子对象，则要把该对象移到该面板的上方，确保当前处于编辑工具，右键并选择“成为底下面板的子对象”。

类似的，如果要脱离父子关系，则选择子对象，同样菜单里选择“从底下面板Unparent”。
 

▪自动为对象设置标签

选择对象，按 "CTRL + L" 快捷键为对象自动设置标签文本。

另一种方法是，在属性编辑器的标签那一栏，点"<"按钮。

 

▪改变对象叠放顺序

选择对象，按 "CTRL + 向上方向键"来将所选对象置于顶层，盖停开其它有交叠的对象。按 "CTRL + 向下方向键" 来将它们置于底层，被其它有交叠对象覆盖。

Picker面板一般处于底层而被其它对象覆盖。

 

▪编辑已存在的图片:

picker对象有多个属性使用到图片，比如面板背景图片，选择按钮，命令按钮滑条滑杠使用的图片等。

在属性编辑器，每个图片路径输入框的右方，都有一个>按钮，点击选择"编辑图片.."来打开图片处理对话框。

 

▪自动设置选择按钮的后续动作

后续动作是指当鼠标点击相应的选择按钮后，是自动切到位移工具还是旋转，缩放工具或是什么也不做。

适当的后续动作可以为动画师省去切换工具的麻烦。

点击　编辑菜单/ 自动设置选择按钮后续动作，　它将会测试所选选择按钮的成员，如果该选择按钮的成员的旋转，缩放属性全部被锁，只有位移可用，则它会切换该按钮的后续动作为"切到位移工具"。
