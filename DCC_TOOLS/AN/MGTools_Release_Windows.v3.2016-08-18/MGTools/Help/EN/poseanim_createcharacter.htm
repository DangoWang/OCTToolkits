How To Create Character:

▪Click the button at the top right of the tool / New Character..

▪Character menu/New Character..

▪If current project has not character built, click the [+ Character..] button in the Hierarchy area.

▪Right click on the character list at the left side / New Character..

 

New Character Dialog:

Character Name: Input a character name, only English letters, digit and under-bar are allowed.

Add Category name: The list shows all the unused category presets.

Multiple selection are supported, when you create the character, the categories highlighted will be added as well.

Custom Category: If the preset categories don't get what you want , input a custom category name and press the "Add to Preset" button to add it to the preset list.

Create: Create the character, also the selected categories.

Cancel: Cancel the character creation.

 

Add Category:

▪Click the button at the top right of the tool / New Category..

▪Category menu/New Category..

▪If current character has not categories built, click the [+ Category..] button in the Hierarchy area.

▪Right click on the character list at left side / New Category..

 

Add Category Dialog:

The top list: Shows all the category presets that has not been used in the current character.

Custom Category: If the preset categories don't get what you want , input a custom category name and press the "Add to Preset" button to add it to the list.

Add: Add the selected categories highlighted in the list.

Cancel: Cancel the categories adding.
