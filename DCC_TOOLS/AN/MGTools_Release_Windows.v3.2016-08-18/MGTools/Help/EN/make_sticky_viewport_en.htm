Make a sticky view:

Create a camera which is constrained to current selection. Not matter how the object move ,in the sticky camera view it will be kept in center.

In the sticky view port ,u can freely rotate it to see different angle of the object.

It could be useful if u want to keep watching some part of a character or when u are checking the intersection.

 

How to do this:

1.Select the object u want to focus on and click the button.

2.A camera which is constrained to the object will be created and current view will automatically be changed to the new camera.

3.Freely change the angle of the viewport. when playing the animation, it will remain the angle.

4.Select the object and click the button again if u don't need the camera anymore.
