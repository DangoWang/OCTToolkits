MGTools:

 

Click to minimize/Restore MGTools. You can custom the size and position of both minimize mode or normal mode of MGTools.

It will automatically remember the size and position information.

Right click on MGTools icon:

MGLAND|Twincodes News.. : Load MG-News standalone program to read the latest news of MGLAND.com.

If MGTools still not know where the program is, it will popup a dialog to let you pick up the MG-News program.

MGTools config.. : Load MGTools configuration dialog to config MGTools hotkey, auto-loading things, installing shelves, etc.

Remember style: Remember current MGTools size and position information.
Style options: There are four Layout styles: single column,  double columns, square and single row.

User style is a style that don't belong to these four style presets.

Pick one of these style options will automatically set MGTools UI size to the corresponding value.

Update/Update MGTools Form Server: Update MGTools from server source.

This requires that current MGTools was installed in server program mode.

If you want to transform current MGTools from local program into sever program mode:

Update/Relocate MGTools Server: Transform MGtools from local program mode into server program mode.

If currently MGTools already in server program mode, use it to pick a new server MGTools source.

This is useful when your MGTools server source is down, missing or deleted.
Beside, if your MGTools was installed as server mode, you could use it to transform to full server mode and vise versa.

Floating Licence: Login/out the floating licence.

Node-Locked Licence / Register MGTools: Open up MGTools register dialog.

Node-Locked Licence / Submit UserCode to Server:

If your MGTools was installed in server program mode, use it to submit your usercode, as well as the IP, userName, to your server.

(Not submit to mgland.com, but to your own server)

The info will be store in .mgreg files in the MGTools_LicenseData folder in MGTools server source.

Install/Uninstall/ MGTools Installer.. : load MGTools shelf installer to install MGTools features to maya shelves.

Install/Uninstall/ MGTools Uninstaller.. : load MGTools uninstall wizard to uninstall MGTools completely.

Help/..: Read MGTools manual and update manuals.

About MGTools..: Know more about MGTools development history, current version and registry state.

Purchase MGTools Pro License.. : Visit mgland.com or twincodes.com to purchase MGTools.

MGLAND animation stuido.. : Visit mgland.com or twincodes.com.

 

Tips: Click button in normal mode will set it to minimized mode and remember the normal size & position information.

Click MGTools button in minimized mode will restore it to normal mode and remember  the minimized size & position information.
