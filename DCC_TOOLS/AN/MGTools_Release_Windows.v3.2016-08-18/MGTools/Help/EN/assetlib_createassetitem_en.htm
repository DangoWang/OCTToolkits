New Asset item:

Press the button on MG-Asset Library UI to open this window:

If you select a reference node then open this dialog, the UI will automatically fill in the information for you.

The items on the UI is pretty straightforward,  here we just talk about a few:

Make Preview Icons: If checked on, the snapshot component will be visible, so the preview images will also be made when creating the asset item.

Notice, only when the asset info comes from the asset that already in you scene, this check box is enabled,

because it only snapshots something that already in your maya scene.

FromFile: Press this button to pick a maya file to read information form.

FromScene: Select a referenced node and press this button to read the reference information.

: Input this asset item data to the category selected in the second row,the project shown in the first row.
