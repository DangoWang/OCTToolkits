Super SnapKeyframe

 

Snap the animation keyframes to integer by force.

If time range is selected before snapping, only the keys in the range are snapped. Or else all animation will be snapped.

 

How to Use：

The tool has no UI, just press the button on keyframeTool or the shelf button on maya shelf.

 

What's different to Maya's snap key feature：

Maya 's default snap key feature can't deal with more than two keys stuck together within one frame.

Super snapKeyframe first does maya's default snapping for one time,than insert a integer key near each non-integer keyframe to keep the animcurve shape,then remove the non-integer key. Hence the animation is the maximal close to the original animation after snapping.

 

More snap option：

Right mouse button clicking on the button,there are more snapping manner:
         

Snap GraphEditor Selected: Only snap the animCurves selected in graphEditor.

Snap TimeSlider Selected: Only snap selected range in timeSlider.

Snap TimeSlider Range: Snap the keyframes between current showRange of timeSlider.

Snap All: Snap all animation of current selected objects.
