Pivot stuff manager:

 

Load pivot stuff lister to manager the pivot stuffs in your scene.

U can easy select / delete pivot stuff there.

 

How to load ( Do one of the following ) :

▪Right mouse button clicking on the button of MGTools and drag downwards and release.

▪Right mouse button clicking on the button or and select "PivotStuff lister"

▪Click icon in your Maya shelves . (If U have installed all the MGTools shelves into Maya shelves)

 

Pivot stuff manager UI:

 

The Left list: Shows the pivot stuff by what they are controlling.

The right list: Show the three pivot stuff controllers  of the pivot stuff.

Tips: Right mouse button clicking on the list left:

select : Select all three pivot stuff controllers who control the object. U can double click it to do the save thing.

Delete: Delete all three pivot stuffs who are controlling this object.

Right mouse button clicking on the right list:

select: Select the pivot stuff controller. U can double click it to do the same thing.
