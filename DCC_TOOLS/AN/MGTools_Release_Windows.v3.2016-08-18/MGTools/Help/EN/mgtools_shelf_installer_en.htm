To load MGTools Installer ( Do one of the following ) :

▪Click in MGTools Config UI.

▪Right mouse button clicking on the first icon of MGTools and select "MGTools installer.."

The Shelf Icons list: The button and checkbox list shows all MGTools features that could be installed to Maya shelves.

Notice that some of the icons maybe disabled, that means the features are not available due to your MGTools registry state.

Which Tab U want to edit : This option determines which MGTools shelves will be installed to / updated / removed from: current selected shelfTab or "MGTools" Tab.

Buttons below:

Turn on the checkboxs which stand for the shelves U want to install, then click Install/Updated selected to actually install them.

On the contrary,click "Remove selected & Disabled" if U wanna remove them from current tab/MGTools Tab.

Click "Select All / None" button to quick select all or select none.
 

Tips: Clicking the icons in the UI above will simply install the shelf that this icon stands for to the current editing shelf tab.If it already exists in the target shelf tab,this action will remove it. That means all icons in the UI above is a shelf toggle for all the corresponding shelves.

U can do shelf management in MGTools Config UI as well.

If you go beyond the trial period and MGTools is not registered nor connected to floating license, there will be some icons disabled, which means not installable.
