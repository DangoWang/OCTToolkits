Efficient Undo/Redo:

 

Click to undo/redo Maya operation. The beauty is that it can step through the null undo/redo and undo/redo many steps by one click.

More option can be found through the option box.

 

How to call it ( Do one of the following ) :

▪Right mouse button clicking on the button of  MGTools,select "EfficientUndo../EfficientRedo.."

▪Click the  icon in your Maya shelves (If U have installed all the MGTools shelves into Maya shelves) .Invoking by this way will only able to undo/redo one step.

▪Click the EfficientUndo/Redo button of the miniToolBox. (or the option box on the right side of it.)

 

When to use it:

1.Want to undo/redo quickly,or the steps need to be undone are too many.

2.Sometime after clicking the playback button in Maya,it ends up with great mount of nonsense undo steps. If U want to undo a real step,U got press z crazily,vice verse.

The Option Box:

 

The undo redo option shares the same UI. Change operation between undo & redo,the UI element will be updated correspondingly . Here we take undo as example:
Undo Times: How many undo steps will be done through one click on the "Undo!!" button of the right corner.
Auto-StepThrough  Null Undo： When doing undo, if it meets a non-sense undo, it will keep undoing until it meets another undo that makes sense. this option is grayed out because nobody want to do non-sense undo one by one..
Auto-StepThrough   Undo With Name：When this option is checked on, input a undo step name.then every time this tool meets the right name, it will automatically step it through.
Query button: Query current undo name and fill the name into the textField left.
Undo!!: Undo by the options been set.
Notice: Unlike maya's optionBox, this option box setting wont be remembered and wont affect the acting of the Efficient Undo/Redo button on the minitoolBox .
