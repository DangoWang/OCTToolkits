Anim-Recorder:

 

Click to load Anim-Recorder tools.

This tools record the world/local transformation of objects, you are free to reapply the record back to these objects later.

It is really powerful tool suggested by Kevin Jackson, Animator from Rhythm&Hue studio.

 

 How to use:

1.Open tool. (By clicking the icon if you installed it or right clicking on the Keyframe tool icon on MGTools main UI and select "MG-AnimRecorder..").

2.Select the objects that you want to record animation. Press the + button on the Anim-Recorder UI to add them into the record list.

3.Setup the record option and click the "Record All !!" button to record. Such as the local objects of your rig.

4.Do whatever change to the animation,such as eliminating the animation of Master control, now appearance of local animation change.

5.Setup the apply option and click the "Apply All !!" button to reapply the information back to the objects. Now appearance of the local animation get back too.

 

UI elements:

 

Mini Buttons:

：Click to switch between the normal mode and Manager mode.

 In the default normal mode, you maintain a record list, do the record/apply action. You can only remove items from the record list but can't actually have their record deleted from scene;

 while in manager mode, which displays all existed record in your scene rather than your current working objects. You can only apply them.

 If you really want to delete the record from scene, use the del key/button in this manager mode.

：Click to add scene selection into the record list. Only available in the normal mode.

  Tips: The less objects you added in, the less time the record/apply action needs.

         If you use this tool to eliminate the animation of the Master control:

                 you only want to added the controller objects into the list, not the controlled objects.

                 For example, if the FK hands are controlled by the root control, you only need the root control in the list.

                 If you do need the FK hands in the record list, please bear in mind that select the root control first and then the hand controls then add them in the list.

                 This will help you get the desired result after you reapply the record back to them.

：Click to remove the selected items from the record list in normal mode. Press the "Del" key will do the same thing.

：Click to actually delete the selected items' record from your scene in manager mode. Press the "Del" key will do the same thing in this mode.

：Click to refresh the list to display all the existed records in your scene. Only available in Manager mode.

：Click to open the tool manual. (The page you are reading :))

The record list:

 Display the currently working list in normal mode while the actually existed records in manager mode.

 Press "Del" key to do the two different remove actions in two modes, double clicking on a single item to select the object in your scene.

The record Option:
          World Transformation: Record the world transformation (Position & Rotation) ,the option you may often use.

          Local Values: Record the Translation & Rotation values rather than the world transformation .

         Record All !!:        Click this button to record all the items in the list.

                         The record will be stored as a empty group in your scene under the "MG_animRecordGrp" hierarchy.

                         It will be saved with your scene so once you have your scene opened, you will have the record available.

         >: Click this button to find more record manner choices. Such as only record the selected items in the list.

          Tips: If you only need transformation within a time range being recorded, please selected the time range in maya's time slider before you do the record.

                 When applying the record, it will only affect frames in the same time range.

The apply Option:
         To Keyframes: Apply the recorded transformation at each keyframe per object. This could be the option you probably want to use.

         To Current Pose: Apply the recorded transformation at current frame per object. In other words, to current pose.

         To Each Frames: Apply the recorded transformation at every frame per object within the time range.

         Auto-Simplify : Only makes sense if you set to " To Each Frame" apply mode. It will auto simplify the "Mocap" like animation data after the apply action.

         Apply All !!:  Click this button to apply the records for all the items in the list depends on which apply mode you chose before.

         >: Click this button to find more apply manner choices.

                 Such as only apply to the selected items in the list.

                 More than that, you can install the current apply action as a shelf button in current Maya shelf tab.

                 You may click these shelf buttons to do the apply action even you delete the record information later on.
