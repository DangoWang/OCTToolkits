Share the Asset Library:

1.The IT manager, make a folder in the server, the folder is only writable to someone who need to manage the global library,

2.while the users only have read access to the folder.

3.Then someone who is responsible to build and manage the global library, let's call him asset leader.

4.Asset Leader build his the local library at this local machine.

5.Asset Leader upload the library to the servers side, also the .assetproj file. This could be done within the tool.

6.Layout artist open the shared project with tool and good to go.

7.The shared global library will already be ready, the local library for the project is still empty ,ready for the layout artist to build his own local library.

8.When there is a update in global library, layout artist just need to reopen the tool.

 

How to upload/download the asset Library:

▪Visit Operation Menu/ Upload/Download, there are lots of uploading/downloading features.

▪Right click on the content area, drag up, there are also the uploading/download marking menus.

Upload/Download Checked Items of Current Category: Upload / download the items checked on in current category.

Will merge the uploaded/download data to the target data. Also it will ask you if you want to upload/download the preview images in the Preview_Images folder.

Upload/Download Checked Items of All Categories: Upload / download the items checked on in all categories.  

Upload/Download Current Category: Upload / download current category. Will prompt to ask you:

▪Update The Category: Will merge with the target category file.

▪Replace The Category: Will replace the target category file.

▪Cancel: cancel uploading/downloading.

Upload/Download All Categories: Upload / download all categories. Same prompt dialog will show up.

Upload/Download Project File: Upload / download the .assetproj file.

Upload/Download All Preview Images: Upload / download the Preview_Images folder in current project folder.
