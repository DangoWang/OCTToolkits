Reset selected channels /attributes:

Reset selected channels /attributes in channelBox.

Selected translation and rotation attributes will be reset to zero , scale attributes will be reset to one, and all custom attributes will be reset to their respective default values.

 

How to do it ( Do one of the following ) :

▪In MGchannelBox, select the channels u want to reset, then click the button .

▪Click icon in your Maya shelves to reset the selected channels in Maya's default channelBox/MGchannelBox. (If U have installed all the MGTools shelves into Maya shelves)

▪Assign a hotKey in MGTools config window. Use hotKey instead.

▪Right mouse button click the button in MGTools UI and drag upward then release.
