Path Tracker:

 

Click to open the path tracker.

It is inspired by Spencer Jones' arcTracker, which was introduced by my colleague. Thank you, Spencer.

I do make a lot of improvements compare to arcTracker, such as more efficient on updating ,and multiple tracking items support, etc

 

 

To load path tracker ( Do one of the following ) :

▪Click the button on MGTools main UI.

▪Click the button in MGTools shelf tab. (If U have installed all the MGTools shelves into Maya shelves)

 

PathTracker UI:

 

 

First Row:

: Click to add scene selection into the tracking list.

: Click to remove the selected tracking items from the tracking list.

: Click to manually update the tracking of the selected tracking items.

: Toggle on / off the linking to the camera. The field right shows the camera name to link.

 Use the "<"  button to link selected camera to the tracking item. If you select no camera before clicking this button,it will add the camera of current active view to be linked.

 The "link" so called, means to track objects' path in the camera view, no matter the camera moves or not, the tracking path will remain still in that view.

: Refresh UI to show the existed tracking item in your scene.

The Lists:

▪Track Target:The list shows all objects to track. Double click the item to manually update that track item.

▪auto:Turn on / off the auto updating condition of the selected tracking items in the tracking list.

▪path:Turn on / off the path visibility of the selected tracking items.

▪fr.:Turn on / off the frame indicator visibility of the selected tracking items.

▪key:Turn on / off the key indicator visibility of the selected tracking items.

Option Area: The title of this area shows the current tracking target name.

▪Time: The time range of the tracking. Use "<" button to set the start & end range to maya's time range.

▪=Range: Update the tracking automatically whenever maya time range changes.

▪Path / Frames / Key  Color: Drag the slider to real time change the color of path / frame / key.

▪Size: Drag the slider to real time change the size of  frame & key.
