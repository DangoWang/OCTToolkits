Constraint lister:

 

Load constraint lister to manager the constraint of the constraint object.

Or click the icon in your Maya shelves (If U have installed all the MGTools shelves into Maya shelves) .

 

Features:

▪Quickly select among the constraint node,the constraint mater and the constrained object to clarify who is constrained by who.

▪Quickly clear the constraint.

▪Correct the weight attribute name.

▪Remove the constraint master object from the constraint master list.

▪Export / Import constraint offset.

 

Constraint lister UI:

Select the constrained object ,then click the button in constraint tool or the shelf in maya shelves.


 

Constraint nodes list : Shows the constraint nodes of the selected object. Double click it to select the constraint node.

Master nodes list : Shows the master list of the constrained object. Double click it to select the master node.

Correct weight name : Correct the weight attribute name. When u rename the master node but the constraint weight attribute name wont update. use this to update it.

Delete constraint: Delete the selected constraint node . U can press Del key in your keyboard to do the same thing.

Clear constraint : Delete all the constraint nodes.

Export constraint offset.. :Export selected constraint node's offset information in the constraint nodes list.

Import constraint offset..: Import offset to currently selected constraint node in the constraint nodes list.

Clear offsetFile..: Delete all constraint offset file exported before in the Maya temp folder .

Remove from targetList : Remove the constraint master object from the constraint master list. U can select the item in the right list and press Del key in your keyboard to do the same thing.

Modify constraint Axis: Modify the constraint axis of the selected constrained object.

Reselect and refresh list : Reselect the constrained slave and refresh the constraint lister list.

Refresh UI : Reread the selected node in your scenes.

Tips : Some features above are only available in the right mouse button clicking popup menu of the list.
