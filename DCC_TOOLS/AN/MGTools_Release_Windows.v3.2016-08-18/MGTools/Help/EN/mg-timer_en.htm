MG-Timer:

 

MG-Timer ,u can use it to record timing in any form: NTSC, PAL, Film, HH:MM:SS ... etc.

U can also use it to calc the frames between different frame rates.

In MGTools 2.0,a calculator is embed into MG-Timer.

 

MG-Timer UI:

 

The LCD screen liked area: It shows the result of recording or converting. The three characters in the left top corner indicates current output unit.

Tips : Right mouse button clicking on this area: U can pick any frame rates and presenting format. This will transform the result into the new format directly.

Left mouse button clicking on the area: A quick way to start / stop recording.

Buttons in the mid part:

Record -> ( Unit ): Click to start / stop recording.

The first button on the left: The unit of your input data. (Right mouse button clicking on it to pick the other units. )

The text field: Put your data into this field to be converted.

->Convert ! Button: Convert the input data to the target unit and show the result in the area above it. (Right mouse button clicking it to pick other unit. )

The scroll field below shows a history note:

All recording / converting result will be kept in this area until u manually erase it.

If it is recording result , if will add a line starting with "Record: ...."

Or it will  add a line starting with "Convert:..."

Tips: Right mouse button clicking on it and select "Clear Message" : Clear all the notes.

Calculator Part:

         Math expression input field: Type in a math expression here as shown above.

Keep in mind that there is no calculation priority here, all calculation goes from left to right. For example, the expression "1+2*3" will get a result  "9", not "7" .

So you just type straightforward and don't worry about the calculation priority.

The "()[]{}" is not supported. The only signs other then digital numbers are ".+-*/%" .

You can type expression in using both the buttons below and your keyboard.

         <- TimerResult: Append the timer result number into input field to join the calculation.

         Other->: Press this button ,choose a operation in the pop up menu. Such as square root, pow ,etc.

 Clear button: Clear the input field.

 BackSpace button: Remove the last character of the expression.

 Other buttons: Press the digital buttons to input a certain number, press "=" button to get the result. The result will be appended to the result history area.

It is often to use right area of your keyboard to do the type in. (Should set focus at the input field first,of course)
