Folder Shortcut:

 

Click the icon on the MGTools UI to open current project folder.Double click to open both project folder and project image folder.

All features of this icon have something to do with "Folder". U can embed your own folder shortcut into it.

 

Right mouse button clicking on the icon:

Menu items above the "open current scene dir.." menu: User embed folder shortcuts will be listed in this area.

Open current scene dir.. : Open the current opening scene folder.

Open maya failSave dir.. : Open Maya failed save directory which temporarily stores ma file when maya crashes.(Hope it is useless)

Open maya script dir..: Open Maya script folder.

Open maya program dir..: Open Maya program folder.

Open maya plugin dir..: Open Maya plug-ins folder.

Open maya preference dir..: Open Maya preference folder.

Open maya icon dir..: Open Maya user icon folder.

Open maya shelf dir..: Open Maya user shelf folder.

Open MGtool folder..: Open MGTools program folder.

 Add custom directory..: Add a user customized folder shortcut.

Edit custom directory..: Edit user customized folder shortcuts.

 Help..: Load Folder shortcut help.

 

Add a user folder shortcut:

Right mouse button click button and select Add custom directory.. to add a user custom folder shortcut.

After that ,a picking folder dialog will pop out.

Pick the folder U want and click OK button :

The text field of the first row: shows the folder path U have picked before.

Name you want to show in popMenu: The name which will be shown in the right mouse button clicking menu of .If it will be installed as shelf ,this will be the label of the shelf.

Shelf Label: If U want to install to shelf ,this will be the image overlay label.

Save to menu Button: Click it to embed the shortcut into MGTools folder icon's right mouse button clicking menu.

add to shelf Button: Add this shortcut as shelf  to Maya shelves.

Close Button: Close add shortcut UI.
