Shelf Convertor

 

 

Convert shelf files between different Maya version.

High maya version tends to add new flags in the shelf file, which makes it unable to be reused in lower version Maya .

This mini tool helps you out :)

 To load Shelf Convertor ( Do one of the following ) :

▪Right click on of MGTools,select "MG-ShelfConverter..".

ShelfConvertor UI:

Buttons:

: Click to pick a folder and scan all the shelf files into the convert list below.

: Click to pick a shelf file and add it into the convert list.

: Remove the selected shelf file items from the convert list. Use "del" key in your keyboard to do the same thing.

The Convert List: Shows all the source shelf files to be converted.

Option Area:

Target Version: Click to select the target maya version to convert to.

Save to the Same Folder: Save the result shelf file in the same folder where the source shelf file is.

To Folder: Click the "Pick.." button to pick a target folder to store the converted shelf files.

Add Subfix : Add a suffix to the result shelf files to prevent the overriding.

Convert !! : Actually begin converting.
