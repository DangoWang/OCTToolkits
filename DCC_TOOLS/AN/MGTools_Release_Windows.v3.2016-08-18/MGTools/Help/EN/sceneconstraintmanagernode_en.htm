In-Scene Constraint Manager Node:

It is just a normal transform node with a nurbs curve shape.

The purpose of the node, is to enable you to collect all the constraint switch attributes, onto one single node,

so you can switch the weights, keyframe them using this one single node.

Actually, you can collect any numeric attributes to this node, not just limit to the constraint switch attributes.

 

Where to find this feature:

▪Right clicking on the constraint tool icon in MGTools tollbar.

▪Right clicking on most tools in Constraint toolbar.

How to use it:

1.Select the nodes that you want to transfer and collect its attributes.

2.In MG-ChannelBox or Maya channelbox, select one or more attributes you want to collect.

3.Use the entries narrated above, input target attribute names and collect these attributes.

4.The generated constraint manager node will be locked on currently activated viewport.

 

tips :After the constraint manager node is created, if you want to lock it to the other viewport,

just active the viewport, use the "Lock ConstraintManager to Active View" to lock the node to it.

If you do not want to lock the node in viewport, use "Unlock ConstraintManager from View";

If you want to scale up or down the constraint manager node, use the "Icon Size" attribute of the node.
