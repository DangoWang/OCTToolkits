File/Program shortCut:

 

Click the icon on the MGTools UI to open a certain file / program .

All features of this icon have something to do with "File". U can embed your own file shortcut into it as well.

 

Right mouse button clicking on the icon:

Menu items above Windows Calculator..: User embed file shortcut will be list in this area.

Windows Calculator.. : Select this item to load calculator of windows to calculate frames & wages :P

Add custom Tool..: Add a user customized file shortcut.

Edit custom directory..: Edit user customized file shortcuts.

Remove default..: Set the button's clicking default  loading  file to none.

 Help..: Load file shortcut help.

 

Add a user file shortcut:

Right mouse button click and select Add custom tool.. to add a user custom file shortcut.

After that ,picking file dialog will pop out,

pick the file U want and click OK button :

The text field of the first row: shows the file path U have picked before.

  Name you want to show in popMenu: The name which will be shown in the right mouse button clicking menu of .If it will be installed as shelf ,this will be the label of the shelf.

Shelf Label: If U want to install to shelf ,this will be the image overlay label.

Set as Default: Set this file shortcut as the default action of clicking the icon on MGTools.

Save to menu Button: Click it to embed the shortcut into MGTools icon's right mouse button clicking menu.

add to shelf Button: Add this shortcut as shelf in Maya shelves.

Close Button: Close add shortcut UI.
