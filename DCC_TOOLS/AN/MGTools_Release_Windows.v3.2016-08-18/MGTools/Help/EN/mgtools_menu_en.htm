MGTools Menu:

To Install MGTools Menu  ( Do one of the following ) :

▪Install it when you install MGTools.

▪Install it in MGTools Config UI.

After successfully installed MGTools menu:

MGTools Menu: All the futures in it could be found in MGTools UI. it exists for saving  your workspace :)

Tips: If U have updated MGTools, Use "Refresh MGTools" to reload the mels that can't be reloaded by reopening the tools.

Actually MGTools Pro3 use a lazy script sourcing mechanism. The script already be sourced won't source again.

So "Refresh MGtools" could be the only way to resource these scripts, beside reopening the Maya.
