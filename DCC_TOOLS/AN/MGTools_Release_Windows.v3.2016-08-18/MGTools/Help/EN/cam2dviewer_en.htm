Cam2dViewer:

 

Click to set current tool to cam2dViewer tool.

Use it to browse your 3D view in 2D way,such as zoom in / out / pan / rotate .

 

 To load cam2dViewer ( Do one of the following ) :

▪Click the button of MGTools main UI.

▪Click icon in your Maya shelves . (If U have installed all the MGTools shelves into Maya shelves)

▪Click icon in your Maya shelves to reset all change made by cam2dViewer tool . (If U have installed all the MGTools shelves into Maya shelves)
 

How it works:

This tool actually real time changes the 2d attributes of a camera to work. such as "post scale","film translate", "file Roll" .

It is really useful when you are doing QC. Animator can use it to zoom in a area to do some detailed work.

In Maya 2011, the tool cursor could be different than the elder maya version, it use the cursor provided by Maya. (Maya 2011, I don't like you ... )

 

How to use it:

▪LMB drag to zoom in / out .

▪MMB drag to pan around.

▪shift+LMB drag to rotate, by default it will auto snapped to 45 degree.

▪shift+MMB reset the rotation.

▪ctrl+LMB to box zoom in / out an area.

▪ctrl+MMB click / Enter key to reset all.

Tool Option: Double click on cam2dViewer icon in Maya's toolbar or right click on the button of MGTools main UI and select "Tool Option.." .        

▪Auto-Reset Camera before save: Whenever Maya save / save as a file, the camera change made by cam2dViewer tool will be auto reset.
Which means if you left this option on, which by default it is on, you won't be able to save the change by cam2dViewer with your scene.
It is design to be helpful in case that you accidentally touch the viewport using this tool.

▪Auto-Snap 45 degree rotation:  Whether or not auto snap the angle to every 45 degree when doing the rotation.

▪Reset Now: Reset all camera change made by cam2dViewer. If the camera attribute is not changed by cam2dViewer,it will not be reset.

         You can use the button on MGTools shelf tab to do the same thing.

Tips: Right click on the button of MGTools main UI, select "Assign HotKey.." to open MG-Config window to assign a hotkey to load cam2dViewer quickly.
