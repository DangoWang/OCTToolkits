Edit file Shortcut:

Click to manage user file shortcut.

 

To edit file shortcut ( Do one of the following ) :

▪Right mouse button click the icon on MGTools UI and select Edit existed tool..

▪Click icon in your Maya shelves . (If U have installed all the MGTools shelves into Maya shelves)

 
Edit file shortcut UI:

Default Action: It shows the default file that will be loaded by clicking the icon of MGTools.

List in section A: Shows the file shortcut list that have been embed by user before.

First column shows the menu item label .the second shows the actual file path ,the third shows the corresponding annotation.

Tips: right mouse button clicking on the list of the section A:

Edit: Put selected item of the list into edit area of section B. U can do this by clicking edit button as well.

Make Default: Set this file shortcut as the default action of clicking icon on MGTools.

Move Up: Move up the selected item one row. U can do this by clicking Move UP button.

Move Dn: Move down the selected item one row. U can do this by clicking Move DN button.

Delete: Delete selected items . U can do this by pressing Del key in your keyboard.

Reload: Refresh the list.

Section B :The edit area of the file shortcut. Select on item and click edit button to open this area.

Menu Name: The name which will be shown in the right mouse button clicking menu of .If it is installed as shelf ,this will be the label of the shelf.

File Path: The path of the file. Click Repick to repick the folder.

Annotation: The annotation of the folder shortcut.

Submit Current Content Edit: Submit the edit to the list of section A.

Apply All Edit: Actually save all edit.

Close: Close edit folder shortcut UI.

  After editing the shortcut in section B, Do remember click "Submit Current Content Edit " button to submit the editing.

If U are satisfied with all the change ,click Save button to save all editing  or it will be accepted as "cancel".
