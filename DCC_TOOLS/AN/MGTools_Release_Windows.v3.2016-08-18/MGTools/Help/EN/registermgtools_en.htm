Register MGTools:

For individual license holder of MGTools Pro, a normal register dialog is preferred.

To Register:

▪Right click on the first icon of MGTools/ Register/Register MGTools..

▪MGTools Menu/Register/Register MGTools..

 

Depending to the registry state of MGTools, the dialog that pops out may be different.
Most of the chance it is a registry dialog for MGTools in trail period:

No matter what does the dialog exactly look like, there will be a text field shows your user code.

Click the copy button at the right side of the user code text field to copy the user code. If this failed, just manually select and ctrl+C to copy the user code.

Go to http://mgland.com or http://twincodes.com by clicking the "Purchase Commercial License" button to purchase MGTools commercial license.

Attach the user codes while paying. If you forget, please resend to mgtoolspro@gmail.com with the payment email box.

You will receive a authorization code for your MGTools.

Paste them in the Authorization code filed and press the "Activate !!" button to register MGTools.

 

Batch Registering MGTools:

For multiple license holders of MGTools Pro, a batch register mechanism may be needed.

Please refer to the "Distribute MGTools" section in the left to get more details.

Please keep in mind that only the MGTools installed in server program mode can use the batch register feature.

To Batch Registering MGTools:
Right click on the first icon of MGTools/ Register/Batch Register MGTools..

MGTools Menu/Register/Batch Register MGTools..
