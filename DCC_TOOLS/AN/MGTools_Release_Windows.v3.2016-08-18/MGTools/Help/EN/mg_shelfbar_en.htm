MG-Shelf:

Click the icon in MGTools to load MG-Shelf bar.

MG-Shelf UI:

 SHF tools button: Similar to MGTools icon in MGTools UI. It is used to manage the UI itself. Click it to minimize / restore Shelf bar.

: Middle mouse button dragging a shelf icon onto it will install the shelf to the current Maya shelf tab.

: Click to load MG-Shelf Manager to add new shelf into MG-Shelf.

: Click to load MG-Shelf Manager to edit MG-Shelf. Middle Mouse button dragging a shelf icon onto it will edit that shelf .

: Middle Mouse button dragging a shelf icon onto it will delete that shelf.

:Save all edit on MG-Shelf. If you close MG-Shelf UI without saving, all edit will be lost.

The area below the top row: They are all user-embed scripts organized in different categories.s

Each frame layout stands for a category. Category can be edited via MG-Shelf manager.

Click the category title to fold/unfold each category.

Middle mouse drag a shelf icon from one category to another category title area if you want.

Drag onto the other shelves button to place that shelf into that position.

Tips: Right mouse button clicking on every shelf button:

               

Edit..：Load MG-Shelf Manager to edit this shelf.

Delete.. : Delete this shelf.

Rename.. :Rename this shelf label.

Rename Overlay label.. :Rename the overlay label of this shelf (The four characters shown on the bottom of the shelf).

Clone : Clone the shelf in place for later use.

Reset Icon to Default.. :Replace this shelf button's icon and highlight icon to the default MG-Shelf icon.

Tips: The tool adds MGTools folder/MGTools_LocalData/MG_UserIcon as its icon search path.

So if your shelf icon is no longer available, the tool wil first search the MGTools local folder,

which is Maya User Script Folder/MGTools/MGTools_LocalData/MG_UserIcon, if the icon with the same file name is found then use the icon instead.

if no yet and MGTools is installed as server program, it will continue to search the MGTools Server Dir/MGTools_LocalData/MG_UserIcon.

So this means, you'd better put the shelf icons into the MG_UserIcon so that it is more portable.
