Workflow of library sharing:

1.The IT manager, make a folder in the server, the folder is only writable to someone who need to manage the global library, while the users only have read access to the folder.

2.Then someone who is responsible to build and manage the global library, let's call him anim leader.

3.The anim leader build the pose and animation library in local side. (He could build them directly at sever side of course, just one more backup)
Make sure when the root dir of the global library, is pointing the prepared server folder.

4.The anim leader upload the library to global side in the tool.

5.The anim leader manually upload the .libproj file, the file is in Maya User Application Folder/MGTools_GlobalData/MG_PoseAnimLib/TheLibProjName/TheLibProjName.libproj.
He could upload the file to any folder that users could catch, or just email them.

6.Users use MG-PoseAnim Library tool open that .libproj, everything is now ready to use. The local library maybe still empty , but the global library are fully ready to use.

7.Done. Users could build up their own pose/anim library at their local side, they could use but can't edit the global library, because they don't the the write access to the global library folder.

8.Once the global library has a update, users could refresh the ui to use the latest library.

 

Upload/Download PoseAnimation library:

There are upload and download menus with the Project, Character, Category menu.

Project/ Upload/Download Current Lib: If currently you are browsing the pose lib of local side, use this to upload the whole local pose lib to server side.

If you are browsing the server library, this become a downloading.

Project/ Upload/Download Anim&Pose Libs: If currently you are browsing local lib, use this upload both pose and animation libs to server side.

If you are browsing the server library, this become a downloading.

Character/ Upload/Download Current Character: Upload/download current character.

Category/ Upload/Download Current Category: Upload/download current category.
Item/ Upload/Download Selected Item(s): Select the items, use this to upload/download.

Tips: If currently you are in Library Management item clicked mode, you can right clicking on the content area, the north marking menu provide you the same entries.
