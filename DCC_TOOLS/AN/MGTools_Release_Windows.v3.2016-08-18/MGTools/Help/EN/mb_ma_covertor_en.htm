MB-MA file converter:

 

Click it and follow the prompt windows to convert MB to MA or vise versa by force.

Sometime maya don't allow u to change file format because of the unknown nodes. Use it to help U.

In MGTools 2.5, a bug is fixed.

 

How to do this:

1.Firstly u got to try the default "save as" feature of Maya. If it doesn't work then use this feature.

2.Click the "MB-MA File converter" button or select the same feature in the pop up Menu of miniToolBox icon of MGTools .

3.It will detective whether there are unknown nodes existed in your scene. If there are, then pops up a prompt dialog to ask for your confirmation (to delete these unknown nodes).

4.If user says ok ,then it will clear all the unknown nodes. (This kind of node are useless in your current scenes, because maya can't recognize them. But make sure they are not the result of unloaded plug-ins. if u don't use any plug-in ,just clear them. )

5.If there are unknown reference node, then it will unload these reference temporarily .

6.A saving file dialog pops up . U can use it to actually save the file in another format.

7.Give u a notice that file converting job is done.
