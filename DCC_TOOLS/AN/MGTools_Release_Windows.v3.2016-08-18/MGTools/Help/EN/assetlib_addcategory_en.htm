Adding Category:

After a asset project is created, you need to add category to it to continue.

A category ,is a easy concept, say we often divide asset into character, creature, props, sets, etc. All these are categories.

 

How to add a category:

▪Category Menu / Add Category..

▪If no category been added to the asset project, press the [+ Category..] button.

 

Add Category Dialog:

The list shows the available category presets. If the category already added to the asset project, they won't appear here.

Custom Category: Input a custom category name if the category presets does not cover all your need.

Use the "Add to Preset" button to add the custom category name as preset.

 
Press Add button to add the selected categories in the list, this will fresh the MG-Asset Library UI.

Press Cancel button to cancel out the category creation.
