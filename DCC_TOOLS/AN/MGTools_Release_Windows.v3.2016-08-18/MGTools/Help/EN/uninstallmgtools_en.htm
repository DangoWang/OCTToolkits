To load MGTools Uninstaller ( Do one of the following ) :

▪MGTools Menu/Install/Uninstall/MGTools Uninstaller..

▪Right click on the MGTools first icon /Install/Uninstall/MGTools Uninstaller..

The uninstallation wizard is pretty straight forward. Set the things you want to uninstall, and press the "Uninstall" button.

If all options are checked, MGTools will be completely uninstalled, including shelves, hotkeys, auto-loading codes, etc.

 

Tips:  The uninstallation will only uninstall the local MGtools , the MGTools server source won't be affected.

For MGTools sever source, you could just manually delete the server folder.
