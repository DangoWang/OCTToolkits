If your MGtools Pro was installed in local program mode and now you want to transform it into server program mode,  or you wanna transform it from normal server mode to full server mode or vise versa, you may need this tool.

If your MGTools Pro server source is no longer available, it is moved, deleted or server is down, or for some reason you want to reassign the MGTools server source directory, use this tool too.

 

1.Click the first icon of MGTools shelf tab: . If the MGTools server source is down, this will be the only icon left in MGTools shelf tab, also this will be the only way to open up the MGTools Relocation dialog. If this is not the case, you can find the same entry in MGTools menu.

2.In the MGTools relocation dialog, click the "Pick MGToolsLoader" button, to pick a file called "MGToolsLoader_WrittenByMiguel.mel",
because this is the main entry of whole MGTools, with it picked the tool may be able to find the whole new MGTools source.

3.If you wanna transform MGTools installation between server mode and full server mode, just turn on/off the "Use full server mode (Copy nothing)"

4.The tool will update the files in local side, and update all the MGTools shelf icons and restart MGTools toolbar.

5.Right click on the first icon of MGTools/Install/Uninstall/ MGTools Installer, install the possible missing shelf buttons of MGTools.
