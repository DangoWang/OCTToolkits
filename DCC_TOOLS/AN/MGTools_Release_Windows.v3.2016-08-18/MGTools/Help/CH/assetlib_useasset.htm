生成MA档：        

 

在主界面上勾选一些要Reference资产项，然后下面方式任选一种来打开生成MA档对话框:

▪点击主界面下方的生成MA按钮。

▪主界面操作菜单/ 生成MA..

生成MA档对话框：

▪左侧列表，列出了勾选上的资产项。在这里还可以通过勾选不勾选来决定，这个资产项是否要reference在该ma档。

▪左侧下方的Ref次数输入框和滑条，可以设置上方选中的资产项reference的次数。

▪右侧区域，会根据左侧列表的设置，呈现不同的内容。
这个区域，实际是模拟，当你用selective preload方式开maya场景时，出来的reference preload对话框的内容。
勾上的项，即将来开这个ma档会直接load进的reference.
没有勾上的，就是会defer loaded的reference。

▪编辑模式：简单模式：只是列出左侧列表的资产项。
高级模式：会真的去读取资产项的maya文件，如果里面又有嵌套的reference，将会以层级方式列出来。
        这个模式比较慢。如果你不需要定制资产项内容的reference defer状况时，不需要这个模式。

▪点击继续按钮，来指定生成ma档的路径，然后生成MA档。

▪点取消按钮，关闭对话框，取消生成MA档的操作。

         

 

其它使用资产库方式：        

 

使用资产项，即以Reference/导入/开档的方式使用这些资产文件。

在2.5版里这些操作将追随maya的选项设置。

事实上，删除所选资产项也与这些操作共享同一个对话框界面，所以集合在一起，以最常用到的Reference操作来说明。

在使用它们前，要先在相应资产项前打勾。然后点击资产库工具上的等按钮，弹出下面对话框：


 

图中第一行说明文字中的"(2次)"代表这个操作将被执行两次，每个资产文件被Reference两次进来的意思。

接下来涉及到几个类别的asset, 在上图中就会出现几列列表。
 

之所以要再出现上面的对话框确认，是因为它会使用当前工程的所有类别里勾上的资产项，而非仅仅当前显示的类别。

并且读档这种事在Maya里有时很费时间，比较缓慢，所以还是再确认下比较好！
不过你也可以勾上"不要再确认"， 这样下次就不会再确认了。如果想要重新让它确认，可以重新勾选上 操作菜单/ 操作前确认。

    如果是开档操作，则次数设置无效，任何时候都只开档勾上的第一个资产B项一次。(连续开档多个档案/开一个档案多次都是无意义的吧。。。)

按继续按钮来真正执行这些操作，按取消按钮来取消此次操作。
