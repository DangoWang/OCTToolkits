MB-MA文件转换按钮功能：

   


直接点击将展开对话，将MB另存为MA，或将MA转为MB。

有时Maya另存格式会失败，这时让它帮你。

2.5版本里遇到一些特殊情况无法另存的bug。

 

用法：
 

1.首先尝试Maya默认的另存功能，如果失效，再尝试这个功能。

2.点击工具箱MB-MA强制转换，或MGTools最后一图标上点右键/相同功能。

3.工具将首先测试场景未知节点是否存在，如果有，则弹出对话框让用户确认。

4.用户如果点确认，则工具将清除这些无法识别的节点。（有的是没安装/加载的插件留下的，有的是文件部分损坏的节点）

5.如果这个未知节点来自外部reference文件，工具将unload该reference.

6.弹出另存对话框，这时应该可以正常另存。

7.提示你转格式成功。
