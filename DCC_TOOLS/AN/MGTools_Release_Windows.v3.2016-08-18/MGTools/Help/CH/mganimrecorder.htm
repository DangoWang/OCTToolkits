MG动画记录器

点击将打开MG动画记录器。

这个记录器可以记录控制器的全局变形信息/控制器本身变形数值。

它的一个主要应用，会是使你的动画在大圈key和生key之前方便互转。

这个工具由Rhythm&Hue动画主管Kevin Jackson建议加入，谢谢 ！

 

典型用法:

1.打开MG动画记录器界面(点击shelf上的(如果你有装)，或者右键点击关键帧工具图标，选"MG动画记录器..")。

2.在场景里选择你想要记录动画的物体。点界面右上角的+号按钮，把它们加入到记录列表中。

3.设置是要记录全局信息还是局部数值的记录选项，点击"记录全部!!"按钮。

4.以大圈(即global层级，用来位移整个角色的大控制器) 动画转为生key动画(即大圈不动，纯用局部控制器key的动画)为例，
先把大圈的动画删掉，然后设置好界面中的应用选项，点击"应用全部!!"按钮。

5.动画转换好了。

 

界面介绍：

界面顶部的图标区：

：点击在一般模式及管理器模式切换。

 在默认的记录模式下，你可以维护一个当前记录的列表，可以记录/应用信息，但只能把物体从当前记录列表里删掉，而不能真正地删除其记录。

 而管理器模式，则可以显示当前场景中已有的所有记录，在该模式下只能应用记录，但按删除图标或del键则真正地删除该记录。

：将场景所选物体依次加入记录列表。
  提示：在选择这些控制器时，如果所选控制器间有父子层级关系或控制/被控制器关系，则要先选父层级/控制层级，然后再选子层级/被控制层级，然后再加入记录列表。
         比如先选髋部控制器然后才选手的控制器。这样将来应用记录才能够正确发挥作用。        

         不过，如果手部控制器完全被髋部控制器控制，则完全不需要把它加入记录列表。

         请只加入你觉得需要加入的控制器到记录列表里。记录对象越少，操作效率越高。

：将列表所选的项目从记录对象中去掉不再记录(一般模式下)。按del键也可以。

：刷新显示场景里所有已存在的动画记录 (管理器模式下)。

：将列表所选的项目的记录实际删掉。(管理器模式下)。按del键也可以。

：查看该工具帮助，即是打开这个页面罗。

界面中间的列表区：

 显示了当前记录的列表(一般模式下)， 或显示场景所有已存在的动画记录(管理器模式下)。

 按del键在不同模式下执行不同的清除动作，双击单项来在场景中选择相应的节点。

记录区：
         全局变形信息：将记录的是控制器的世界坐标变形信息。应用地比较多。

         控制器属性值： 将记录的是控制器的本身的位移，旋转信息。

         记录全部！！：点击该按钮，则记录所有在记录列表中的物体。

                         该记录将以空组的形式( 请不要看成是" 空姐") 存在于当前场景，在"MG_animRecordGrp"这个层级下。

                         如果没有清除则只要打开该场景即可以用到。

         > ：点击该按钮，会弹出上图的菜单，来只记录在列表中所选的物体的变形信息。

          提示：如果你只想记录从某帧到某帧的时间范围内的变形信息，则在点击记录按钮前，先在时间滑条上选一段时间范围，则记录只限于该范围内。将来应用也将限于该范围。

应用区：
         到关键帧： 将先前记录的变形信息应用回每个控制器即有关键帧上。应用地比较多。

         到当前Pose：将先前记录的变形信息应用回每个控制器的当前帧上。即是以Pose方式应用。

         到每一帧：将先前记录的变形信息应用回每个控制器每个帧上。如果没有勾上自动清理帧，则密密麻麻全是帧。

         自动清理帧：只有应用模式在"到每一帧" 下才有效。它可以自动简化生成的逐帧动画。

         应用全部！！：点击该按钮，则将记录应用回列表里所有的控制器物体。

         > ：点击该按钮，会弹出上图的菜单。可以选择只应用到所选的控制器。

                 也可以把上述两个应用动作，作为工具架按钮安装到当前Maya的工具架上。

                 即使记录被清除，也可以按这些按钮使用之前的这些记录信息。
