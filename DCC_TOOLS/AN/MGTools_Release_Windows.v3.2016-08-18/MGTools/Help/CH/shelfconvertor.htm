Shelf版本转换器功能：

 

 

shelf版本转换器用来使Maya的shelf文件在各个maya版本间转换。

因为高版本的maya的shelf文件里有一些新加入的标签，导致低版本的Maya无法加载这些shelf。

这个工具可以使之顺利转换，使低版本的Maya也可以用那些从高版本Maya那拷来的shelf文件。

 

启动方式：

▪在MGTools主界面的MG工具架按钮上点击右键/Shelf版本转换器..

▪在MG菜单栏菜单里选迷你工具/Shelf版本转换器..

Shelf版本转换器界面：

第一栏 ：

: 点击拾取一个文件夹，并扫描其包含的shelf文件进列表。

: 点击来选取一个shelf文件并添加到转换列表。

: 将列表所选的shelf项从要转换的列表里删掉，也可以选列表项再按del键来做同样的事。

列表区：显示了目前要转换成其它Maya版本的shelf文件列表。

 选项区：

目标版本：点击选择要转换到的低Maya版本。

存到相同目录下：转换后直接存在与源shelf文件同样的目录下。如果没有添加后缀，则是覆盖掉源文件。

存到目录：点击后面的拾取..按钮来选择一个目录，转换后的shelf文件将会放到那里。

添加后缀：为转换后的文件添加后缀名以防覆盖掉源文件。

转换!!按钮：将列表里的shelf文件转换成目标Maya版本。
