Reference工具箱：

 

源：决定当前Reference操作对象，来自场景所选的对象，还是Reference编辑器所选项目。

复制/粘贴 Reference：选择Reference角色的一部分，或在Reference编辑器里选中一些项，
                               点复制按钮将选中的Reference信息复制。
                                这样在任何一个Maya, 点击粘贴，则会叫进原来复制的Reference角色
                               这里的复制粘贴，只会保证Reference路径一样，namespace尽量一样。

导出为MA档：选择Reference角色的一部分，或在Reference编辑器里选中一些项，
                               点击导出为MA档，即所选的Reference会被导出为maya MA档。

从MA档读取..：点击，则浏览拾取到一个MA档，读取它的Reference信息，并显示类别像Maya Preload窗口一样的窗口，你可以选择一些Reference，叫进到当前场景。

Reference操作：其它的Refence操作，这些操作有的Maya也有，只是因为工具箱可以基于场景所选对象，所以更方便。    比较不同的是修改Namespace：

修改nameSpace：把场景所选的reference的nameSpace用新的名字强制替换掉。如果目标nameSpace被占用，将清除占用它的节点。

Reference信息：把以/或\的方式，显示Refence对象的文件路径。也可以浏览其目录。用法是在场景里选择Reference角色的一部分，或在Reference编辑器里选中一些项，然后点击获取按钮。
