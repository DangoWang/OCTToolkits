插入层级功能：

   


直接点击将弹出对话框提示要创建多少个层级。

用来在两个层级间插入无限个层级以便动画。

 

启动方法：

▪点击在工具架上点图标（如果你有装全MGTools工具shelf）

▪在MGTools工具条上右击，在右键菜单选择"插入层级.."

▪直接在迷你工具箱界面中点"插入层级.."按钮。

 

使用时机：

1.Rig已有的某层级在动画阶段中不够用。

2.该层级至少位移/旋转可被约束。

使用方法：

点击相应按钮弹出如下对话框：

 
在对话框里输入大于3的数字即可按OK按钮来插入新的层级。不能输入小于三的数字，应该那样没有意义。
  注意在生成新层级后，原先所选的那个物体将被新层级约束，所以它不会用来被Key动画；

 如果生成3个新层级，则新层级最顶层被原先所选物体的上层约束，新层级的最底层将约束原先所选物体，新层级里的下面两个层级都可以用来key动画；

 如果想要更多的层级，则直接选这两层级任一个无限打组即可。（因为它不是Reference的）

生成后的新层级将控制原先所选物体，而这个新层级也将被原先所选物体的上层所带动。
