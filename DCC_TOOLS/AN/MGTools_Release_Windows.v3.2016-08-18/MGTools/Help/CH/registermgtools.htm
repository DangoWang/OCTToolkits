注册MGTools

对于个人用户，可以直接使用普通的注册窗口：

访问方式： 右键点击MGTools第一图标/注册/注册MGTools..

                 或者点MGTools主菜单/注册/注册MGTools..        

针对您的MGTools Pro3当前状态，弹出来的注册窗口外观会有所区别。

比较常见的是试用期间，在Maya启动时，会隔一定天数弹出的窗口：

 
不管这些注册窗口的外观，它们都会有一栏显示了您当前计算机的用户码。
点击像上图右上角的复制按钮，将这个用户码复制到剪贴板。如果这个按钮失效，请手动复制。

然后去http://mgland.com或http://twincodes.com网站，购买MGTools相应使用机子数量的商业授权。

购买时请提供这些用户码，以便米高发送激活码，将试用期的MGTools转为商业授权的注册版。

 

批量注册MGTools

    对于企业用户，可能需要对线上几十台机子进行批量注册。        

    请参考 分发MGTools这一栏的帮助，在线上分发安装MGTools.

    安装后，在其中一台机子使用批量注册MGTools.    

访问方式： 右键点击MGTools第一图标/注册/批量注册MGTools..

         或者点MGTools主菜单/注册/批量注册MGTools..        

         如果当前机子的MGTools并不是以服务器模式安装的，则这个功能不可用。

批量注册MGTools的具体帮助，见分发MGTools这一栏。
