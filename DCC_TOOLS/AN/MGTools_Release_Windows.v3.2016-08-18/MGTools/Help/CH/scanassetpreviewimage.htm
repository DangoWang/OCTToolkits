批量扫描资产预览图：

按资产库界面上的 按钮即可打开"从文件夹批量扫描资产预览图"的对话框:

 

预览图扫描及自动与资产项匹配选项：

要扫描的图片格式：要扫描的图片文件格式限定。没勾上的格式不会出现在结果中。

自动匹配条件：预览图与mb/ma档的文件名匹配条件。

▪DB：如果maya档案叫donky.ma，找到donky.jpg就会与之匹配。

▪DB*：如果maya档案叫donky.ma，找到donky_01.jpg就会与之匹配。

▪*DB：如果maya档案叫donky.ma，找到preview_donky.jpg就会与之匹配。

▪*DB*：如果maya档案叫donky.ma，找到preview_donky_01.jpg就会与之匹配。

▪自定义：双击自定义后的输入框即可输入自定义的匹配表达式，支持*号作为通配符。
如果是Maya2011，不用双击即可输入。

扫描子目录：扫描拾取的目录的子目录。

      扫描深度：搜索文件夹的深度，只搜索当前文件夹意味着深度值为1.

同名则只用最新：如果在不同目录下搜到同样名称的图片文件，则只列出修改日期最新的档案。

拾取目录并扫描：点击将弹出拾取目录的对话框，拾取一个目录按确定后将开始扫描，然后把扫描结果放到此行上面。

         类似扫描资产项的操作方式。

         扫描结果的右边部分是预览图的预览。类似资产库的预览图模式。

：将当前扫描出来的有勾选上的预览图路径数据加入各匹配到的资产项的信息里。

    注意，这种方式找到的预览图，其路径最好是线上所有人都可以访问的路径，否则当您把资产库上传后，layout人员还是看不到预览图。
