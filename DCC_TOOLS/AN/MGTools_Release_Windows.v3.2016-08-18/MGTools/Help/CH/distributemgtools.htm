如果您只是在单个机子上安装MGTools，请见左侧的"安装MGTools" 栏。
 

以固定授权方式分发MGTools, 意思是在整个动画生产线里，在多个机子上安装MGTools, 为了安装，注册方便，后期维护，更新方便，采用下面步骤：

1.到http://mgland.com下载MGTools压缩包并解压到服务器上一个线上所有机子都能读取的路径；

2.在压缩包解压的目录里，有一个InstallMGTools_InstallOptionPresets.txt的文件，用写字板打开这个文件;

3.这个文件是你用这个安装包安装时的预置选项，对于在多个机子上分发MGTools,
最好按说明更改里面的 "> Install Mode: " 选项，将其改为 "Install MGTools As Server Program" 为默认选择。

4.保存这个配置文件;

5.确保MGTools目录里的MGTools_LicenseData这个目录，对于所有机子的用户都可写。（其它的只需要可读）

6.在每台机子上拖动该服务器安装目录里的InstallMGTools_WrittenByMiguel.mel到运行中的Maya 3D 视窗并松开。

7.一直点下一步，直到安装结束。这一步可以不用再设置安装选项，因为之前在InstallMGTools_InstallOptionPresets.txt里设置过了。

8.如果安装源的MGTools_LicenseData目录可写，安装后，会自动提交安装机子的用户码，IP， 用户名等信息到该目录，以便软件管理员批量注册。
如果过后发现有机子没有提交，则在那台机子上选择MGTools第一个图标/ 注册 / 向服务器端提交用户码..  即可提交。

9.所有机子装完后，软件管理员，在其中一台装完MGTools机子上右击MGTools第一个图标/注册/批量注册MGTools...

10.在提示身份警告后，弹出下面窗口：

11.假设上面我们安装了3台机子，则如上图所示，区域1显示了三个用户的信息。
如果上面显示的信息少了，则要在少掉的机子上右击MGTools第一个图标/注册/向服务端提交用户码..  来补交信息。

12.点击上图界面中的复制按钮，来复制区域的用户码。如果这一项失灵，则直接全选区域2的用户码信息，ctrl+C复制。

13.在mgland.com上购买MGTools。

14.购买时，或购买后附上上面的用码码信息，米高将生成相应的激活码，这些激活码的文本格式将是：
用户码1 >  相应激活码1
用户码2 >  相应激活码2
用户码3 >  相应激活码3
。。。

15.将上面收到的信息保持上面的格式情况下复制，点击上图中的"2.粘贴并匹配并批量注册"栏，将复制的信息粘贴到下方的输入栏。

16.点击最下方的 "2.匹配 ->"按钮，粘贴入的激活码信息将自动跟区域1不同的用户信息匹配并填入区域1 各自右方的输入栏。

17.如果激活码是对的，则区域1最左的图标将显示为绿色的勾。否则仍是一个红字的禁止符号。

18.点击"3.应用注册!! " 按钮，将这些激活码信息实际写入MGTools_LicenseData目录下各个文件。

19.这样每台机子重启下Maya，MGTools就会是已注册的。这个动作将会把服务这些注册信息应用到本地。

20.本地机子一旦注册完，即使将服务器上的MGTools_LicenseData目录下的注册信息清除，也不会影响本机的使用。
 

以浮动授权的方式分发MGTools: 基本上沿用上面的1~7步骤，要使用浮动授权，请查看这里。

 

按服务器端模式安装的MGToos，更新MGTools的步骤：

1.到http://mgland.com下载最新的安装包，解压覆盖服务器上的MGTools。

2.如果是以完全服务器模式安装，则只要让动画师重启maya即可。完成。

3.如果以一般的服务器模式安装，则需请每一台机子的动画师，右击MGTools第一个图标/更新/从服务器端更新MGtools..

4.完成。

5.这些步骤的原理是，先更新服务器端的MGTools, 然后每台用户机用服务器端更新的功能，来更新本地的MGTools.
