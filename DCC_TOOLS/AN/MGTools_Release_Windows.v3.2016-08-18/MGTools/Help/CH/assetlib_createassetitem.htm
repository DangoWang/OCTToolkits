新建资产项：

按MG资产库右上角的 按钮即可打开新建资产项的对话框:

如果你在打开新建资产项对话框前有在场景里选择一个Reference角色的任一部分，则对话框会自动把该Reference信息读进界面。
如果没有，而当前场景并非未命名未保存过的场景，则会把当前场景的信息抓进来。

界面上的各信息项不难理解，这里说明一些需要解释的：

创建预览图：勾选上的话，会出现生成预览图的界面。这样点导入按钮时，会顺带生成预览图。
    注意，如果你是用读自文件按钮，抓取maya当前场景之外的场景，则这一项不可用。因为它只会就当前打开的maya场景来生成预览图。

读自文件..：点击会弹出指定Maya档案的对话框，选定一个Maya档案即可把其信息读进界面。

读自场景..：在场景里选择一个Reference角色的任一部分，再点击它即可把该Reference信息读进界面。

导入..：将当前资产项数据加入资产库当前项目相应类别的信息里。
