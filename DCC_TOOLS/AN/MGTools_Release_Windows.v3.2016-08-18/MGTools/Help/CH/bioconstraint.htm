互相约束功能：

 

用Maya默认的节点生成一个装置系统，让两个物件产生互相约束的关系.

 

访问方式：

▪在MGTools主工具条的约束工具图标上右键，可以访问到。

▪在约束工具条上点击图标。

 

用法：

1.选择要互相约束的两个变形节点，即可被约束及约束的两个对象。先选择的对象将先默认为约束体。

2.点击按钮，生成互相约束的装置。

3.场景里两个物件的轴心位置，分别生成两个十字架。这是用来key帧及切换约束关系的控制器。
 

提示：先选择对象那边的新控制器，会有一个属性BioConstraint_Weight，这个为1则为当前控制器约束另一控制器，0则表示被约束。

两个新控制器上都有一个示意性的属性： BioConstraint_Master, 值为Yes则表示当前为约束体，No则表示当前为被约束体。

 

  互相约束在实际动画流程中要注意的：

1.避免权重切换的跳动： 要避免在切换互相约束权重在切换时的跳动，可以选择两个十字架控制器，在要切换的那一帧同时set key。这样再切换就不会跳了;

2.不用担心原先层级： 如果两个初始选择对象，都有上层级，则生成的控制器也会受其上层级控制器;    

3.生成控制器的通道栏颜色： 默认会是绿色，因为在第-1帧有key一个帧。请保持这个帧的存在，这样在切换时才不会产生奇怪的递进跳动现象。

4.如果想让物体脱离约束关系： 原始所选的两个物件，被加上的都只是正常的父子约束，你可以像对待正常约束那样，切换相应权重，让其脱离这种互为约束的关系。
