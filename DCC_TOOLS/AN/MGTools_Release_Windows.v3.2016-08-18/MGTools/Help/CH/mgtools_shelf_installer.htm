工具架安装器启动方法：

▪在MGTools配置窗口 点这个按钮，将弹出MGTools 工具安装器窗口。

▪在MGTools工具条第一个图标上点右键/ 安装/卸载 / MGTools工具安装器..

图标列表 ：显示所有提供的可被安装成Shelf的MGTools功能项; 其中可能会有一些图标是灰掉的，这说明这些图标代表的功能对你当前MGTools授权状态来说不可用。

你要编辑哪个工具架 ：这个单选框用来决定是对当前工具架还是对MGTools工具架进行操作。

最底下按钮区：

选择你要安装的shelf，然后点安装/更新所选按钮来安装或更新现有的所选Shelf.

反之想清理则按 "移除所选 & 不能用" 按钮，它将移除勾选 上的，及对于当前MGTools授权不可用的图标。

可以点击全选/全不选按钮来快速全选或全不选。

提示：在每个图标上面直接点击，若当前工具架/MGTools工具架已安装该Shelf,则删掉该Shelf，反之则安装。

                 也就是每个图标都是各自Shelf的开关。

                 你也可以在MGTools设置窗口进行一些Shelf安装工作。

提示：如果目前MGTools已过试用期并且未被注册，则很多图标会被禁用并不被安装。
