标记器功能：

   


直接点击将在所选物体的轴心处生成一个locator。这个locator将同时记录下生成时所在的帧数信息。

也可以在选定帧范围的关键帧处逐个生成标记，也可以多选好几个物体逐个标记。

 

启动方法：

▪点击在工具架上点图标（如果你有装全MGTools工具shelf）

▪在MGTools工具条上右击，在右键菜单选择定位器。

 
使用方法：

1.在场景中选择你想要标记的层级(可多选)，点击相应按钮/菜单标记即可。

2.如果你在时间滑条上选了一段的时间段，则会被视为要根据所选物体的关键帧来建立标记。

3.你可以打开标记器管理器，来利用好这些标记。

标记管理器使用方法：

启动方法：

▪点击在工具架上点图标（如果你有装全MGTools工具shelf）

▪在MGTools工具条上右击，在右键菜单选择定位器管理器..

 

使用方法：

 

列表显示了场景中的标记器。定脚过程中也会自动产生标记器。如果场景中无标记器，会提示无标记器，确定后即关闭。

单击列表项目，则在场景中选择相应的标记器。

双击列表项目，在选择标记器的同时，跳到产生标记器的时间点。因此这些标记器本身位置记录了物体在该帧的位置，它也可以记录下当时的时间点。

重加载按钮：重新读取场景中的标记器。

删除所选项目:删除列表在所选的标记器项目。也可以直接按键盘的Del键。

删除所有项目：删除全部标记器。如果这些标记器有用来做约束或跟外部有连接，则不会被清掉。
