共享 Pose动作库流程：

1.由系统工程师，在项目服务器上生一个目录，这个目录只有将来需要维护全局库的人才有写的权限。其它动画师们都只有读的权限。

2.负责建立和维护全局库的人，（我们暂且叫他动画leader）建立他本地的pose库及动作库。（也可以直接在全局状态下直接建立全局库，不过这样少了备份）

3.动画leader在工具里上传他建立的库到全局。

4.动画leader手动上传他本地maya用户目录/MGTools_GlobalData/MG_PoseAnimLib/相应库工程文件夹/库工程名称.libproj
这个库工程文件上传的路径可以随意，只要大家都可以访问到。甚至可以直接用邮件发流程中的人员。

5.动画师在工具里，用打开工程方式，打开这个被共享的libproj文件。

6.这样就可以开始使用了。动画师可以用本地库建立和使用自己的库，也可以使用被共享的全局库。

7.当全局库有更新，每个动画师重开下工具就可以立即使用到。

 

上传/下载pose动作库:        

在工具菜单栏的库工程，角色，类别，项这几个菜单里，都有上传/下载相应级别数据的功能。

库工程 / 上传/下载当前库：如果当前是pose库则上传或下载pose库。如果当前是动作库则上传或下载动作库。

库工程 / 上传/下载Pose&动作库：同时上传或下载当前库工程里的pose库及动作库。

角色 / 上传/下载当前角色： 将当前库的当前浏览角色上传到全局库或下载到本地库。

类别 / 上传/下载当前类别： 将当前库的当前浏览角色的当前类别上传到全局库或下载到本地库。

项 / 上传/下载所选项：将当前选中的项上传到全局库或下载到本地库。也可以切换到库管理模式，通过在内容区域上点右键也可访问到这个功能。
