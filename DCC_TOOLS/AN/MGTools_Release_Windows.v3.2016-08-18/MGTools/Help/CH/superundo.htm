高效率Undo/Redo功能：

直接点击将撤消/重做一步Maya操作。

不同的是，如果它遇到空的无意义的Undo/Redo，它将一直跳过，直到遇到有意义的Undo/Redo项。

通过在迷你工具箱里点击选项按钮，可以执行更高级的快速Undo/Redo。

 

启动方法：

▪在工具架上点图标（如果你有装全MGTools工具shelf）。直接点这个只能撤消/重做一次有效的Maya操作。

▪在MGTools工具条上右击，在右键菜单选择"高效率Undo.."/"高效率Redo.."

▪直接在迷你工具箱界面中点"高效率Undo.."/"高效率Redo.."按钮。

 

使用时机：

1.想要快速地Undo/Redo，需要Undo/Redo的次数相当多。

2.有时在场景播放动画后，出现一大堆无意义的空Undo项，使每个操作的Undo/Redo变地很繁琐。（有时Key动画就是会碰到啊T_T）

使用方法：

直接点击相应按钮/菜单，即可撤消/重做一步Maya操作，如果碰到无意义的空Undo/Redo项，则直接Undo/Redo到下一个有意义的Undo/Redo项。

点选项按钮弹出高效率Undo/Redo的高级应用界面：


 
在这个界面可以执行更多选项的Undo/Redo，解释如下：

操作: 选择你是要Undo/Redo。界面/功能将随之更新。

提示：下面的说明将以Undo操作为例，如果是Redo操作将是基本一样的。
Undo次数: 填入你希望按右下角Undo !!按钮一次所执行的有效Undo次数。
自动跳过空的无意义Undo项： 在Undo时，如果遇到空的无意义的Undo项，则一口气Undo到下一个有意义的Undo项，然后再根据Undo次数的设定来决定接下来Undo几次。该选项是默认勾选并不可变的。（没有人愿意一步步做无意义的Undo吧。。）
"Undo项带有字符"：勾上该选项后，在后面输入框输入一个Undo名字（如果不知道可以点右边的查询按钮），则每次执行Undo !!时会自动跳过该类型的Undo(即顺带undo掉该undo项)。
查询按钮: 查询最近的Undo名字并填入左边的输入框以便后面做自动跳过此类Undo的操作。
Undo!!: 根据选项设置来执行Undo操作。
提示：并不像Maya的选项设置，该设置只对该界面的 "Undo!!" / "Redo!"! 按钮起作用，而不会改变外面的"高效率Undo/高效率Redo"的执行选项。外面的按钮永远只会undo / redo一次有效maya操作。
