文件夹快捷方式按钮功能：

   

 

直接点击即打开当前工程目录，双击将同时打开工程目录及工程目录下的image目录。

这个按钮主要代表快速访问文件夹功能，可以在里面内置文件夹快捷方式。

 

右键菜单功能：

打开当前场景文件夹..以上区域 ：用户内置的文件夹快捷方式将以菜单命令方式内置在这一区域的菜单中；

打开当前场景文件夹..：在Windows资源管理器打开当前打开场景所在的目录；

 打开当前所选 Reference文件夹： 在场景里选择Reference对象的任意一部分，再执行这个工具，

                                 则打开Reference文件所在文件夹。

打开Maya崩溃暂存文件夹 .. ：打开Maya崩溃时另存档案的目录。（但愿无用）

打开Maya脚本文件夹..：打开我的文档中Maya的script目录。

打开Maya安装文件夹..：打开Maya的安装目录。

打开Maya插件文件夹..：打开Maya的Plug-ins插件目录。

打开Maya配置文件夹..：打开Maya的用户配置目录。

打开Maya图标文件夹..：打开Maya的用户图标放置目录。

打开Maya工具架文件夹..：打开Maya的用户shelf放置目录。

打开MGTools本地文件夹..：打开MGtool的本地的安装目录（在我的文档中相应版本Maya的script目录里）。

打开MGTools服务器文件夹..：如果当前MGTools是以本地模式安装的，则这一项不可用。

如果是以服务器模式安装的，则是打开当前MGTools所指向的服务器上MGTools源目录。

如果想将本地模式安装改为服务器模式安装，可以用安装脚本重新安装一遍，

安装时选择服务器模式;

也可以用MGTools Relocation工具，来变为服务器安装模式。

 添加自定义文件夹..：添加一个用户文件夹快捷方式。

编辑自定义文件夹..：编辑用户内置的文件夹快捷方式。

 帮助 ..：查看快捷文件夹的帮助。

 

添加用户快捷文件夹：

在上点右键/添加自定义文件夹..即可开始添加用户快捷文件夹。

点击后，会弹出拾取文件夹窗口，

拾取你想要的路径后点确定，会弹出新建文件夹快捷方式窗口：

第一栏：显示你之前所拾取的目录。

要在右键菜单里显示的名字 ：将来内置在右键菜单里显示的名字，如果存成shelf就是shelf的标签。

工具架标签：如果要存成shelf，这一栏的文字将作为shelf图标上显示的名字。

保存到右键菜单按钮：将这个文件夹快捷方式内置于MGTools的右键菜单中。

安装到工具架按钮：将这个文件夹快捷方式作为shelf安装在工具架上。

关闭按钮：关闭这个对话窗口。
