中间张工具功能：

 

用来在当前帧前后两关键帧间按一定百分比在当前帧加入BＤ中间张／极张。

 

典型用法：

1.打开帧工具箱，点开中间张工具界面如下图。

 

2.在场景里选择要改变时间间距的物体，并把当前帧放到想要加入BＤ张的时间点。

3.点头排任何一个分数按钮将生成ＢＤ张。

界面选项的解释：        

头排的分数按钮 ：将在两关键帧间，按照所点按钮的百分比数加入BD中间张。

                 左边的是靠上一帧的相应百分比数加入中间张，右边的则是靠下一帧的相应百分比数加入中间张。

自定义百分比按钮 ：用中间输入框来改变自定义百分比的分母。也可以拖动滑条来改变。

                 左边的是靠上一帧的相应自定义百分比数加入中间张，右边的则是靠下一帧的相应自定义百分比数加入中间张。

->BD选项和->极张： ->BD张： 在前后关键帧的数值区间内生成中间张;

                         ->极张：在前后关键帧的数值区间外生成极张;有点像生成逆向预备/逆向缓冲的极张。

       自动避免逆向缓冲：在生成BD张时，通过修改BD张的关键帧曲率避免overshoot逆向缓冲。

       只处理所选动画曲线：只在曲线编辑器里所选动画曲线上生成中间张/极张。

只处理所选关键帧：只处理曲线编辑器里所选关键帧的key（按设定的百分比来改变这些Key数值）。
