固定物体按钮功能：

   

 

直接点击把场景当前选择固定到视图中不动。

比较常用到的是，把角色Rig的表情控制面板选中后固定在视图中以方便调节。

 

用法：

1.在场景中选择你想要固定的层级，点击按钮即可。

2.会多生成一个locator，用这个locator来移动你的物体到想要的角落。

3.如果所选的层级可以缩放，这个locator的缩放将直接缩放该层级。 用locator的offset属性来偏移。
 

固定物体选项窗口：


 

这个选项窗口决定了，是把所选物体的轴心对准视窗，还是把物体的视觉中心对准视窗。

有时角色的表情控制面板轴心在原点，面板却在角色脸部附近，这时可以选视觉中心单选框再点固定按钮进行固定，这样表情面板,移动用的locator与视图就对齐了。

注意只要更改了选项(只要改变了单选框选择就会自动保存这个选项)，下次在外面执行固定物体都会根据这个选项来，即使重启maya也一样。
丢进新层(在下面选层颜色)：把被固定物体及固定用locator丢进一个新层便于管理。

记住当前变形信息：选中被固定物体，或固定用的locator，然后点击该按钮来记住固定用的locator的位移，旋转，缩放，偏移，大小，形状位置等信息。
