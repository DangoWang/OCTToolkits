轴心装置管理器：

直接点击将打开MG可动画轴心装置管理器。

用它来管理场景中所有的轴心装置，当轴心装置建多时这就显得必要。

 

启动方法：

 

▪在MGTools的上点右键并下划。

▪在轴心工具条的中间或图标上点右键/轴心装置管理器..

▪直接点工具架上的图标(如果你有装全MGTools的Shelf)。
 

轴心装置管理器：

 

 

左列表 ：显示轴心装置控制的对象，以此分类各套轴心装置。

右列表：显示左列表所选被控制对象所对应的轴心装置的三个控制器。

提示：如上图在左列表中点右键：

选择: 选择该被控制物体对应的轴心装置，双击列表项目也能选择。

删除: 清除被控制物体的轴心装置。
