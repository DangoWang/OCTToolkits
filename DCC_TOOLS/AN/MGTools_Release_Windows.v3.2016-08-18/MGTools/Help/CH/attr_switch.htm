无跳动属性切换功能：

通过一些设定，即可无跳动切换一些会导致跳动的属性。

用在切换Rig的跟不跟随属性时很好用。( 用在IKFK里的话。。。看Rig的设定，不一定有效。)

 

调用方式：

▪点击在工具架上点图标（如果你有装全MGTools工具shelf）

▪在MGTools的图标上点右键，选无跳动属性切换..

 

典型用法：

1.打开无跳动属性界面如下图。

 

2.在场景里选择包含要切换属性的物体。然后在Maya通道栏或MG通道栏里选择想要切换的属性。

3.点击从通道栏获取按钮，将该属性填入第一行要切换的属性的输入框。(如上图)

4.在要切换的属性值里填入想要切换成的属性值。如果是那种1，0切换属性，可以用从属性猜按钮来猜得你要切换的目标值。

5.在下面要消除跳动的物体这一栏中，用+号按钮来加入场景选择到列表中。
这个列表里，应该放入等这个属性切换后，正常会产生跳动移动的对象。即需要吸附的物体。

6.点击切换！按钮来完成切换。

 

工作原理：

正常一个属性切换，有时会导致自身/别的对象有跳动移位的现象。

这里只是以一个自动的方式，你只需要提供相应的信息，这个工具即可以在当前帧和当前帧的上一帧位置，对所有需要消除跳动的物体自动设关键帧，在当前帧切换属性到目标值，然后自动对所有需要消除跳动的物体进行位移/旋转上的吸附。

 

一些界面按钮的解释：        

+号按钮 ：将场景所选物体加入需要消除跳动的行列。

垃圾箱按钮 ：将列表里所选列表项从需要消除跳动的行列里删除，你也可以在列表里选中相应项，直接按键盘的del键删除。

帮助按钮：打开本页面。
