卸载MGTools：

▪在MGTools工具条第一个图标上点右键/ 安装/卸载 / MGTools工具卸载器..

▪主菜单栏/安装/卸载 / MGTools工具卸载器..

弹出的向导很简单，点击卸载，提示警告后就会开始卸载。

如果全部勾选，则MGTools在本地会被干干净净地被卸载。

提示：这个卸载，只会卸载本地的MGTools, 即本地Maya用户脚本文件夹里安装的MGTools, 及相关的如工具架，快捷键等。

                 另外，即使MGTools是以服务器端模式安装的，这个工具也不会删除服务器上的MGTools源。

                 服务器上的，你可以直接手动删除相应目录即可。
