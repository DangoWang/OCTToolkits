基于帧创建曲线功能：

直接点击将根据所选物体的关键帧，来绘制一条曲线。

可以配合路径动画使用，先简单block出关键帧，再点它生成曲线，再用曲线做路径动画。

 

启动方法：

▪点击在工具架上点图标（如果你有装全MGTools工具shelf）

▪在MGTools工具条上右击，在右键菜单选择描绘动画路径..

 
使用方法：

1.在场景中选择你想要标记的层级，点击相应按钮/菜单。

2.会弹出选项窗口：

 

绘制方法：选关键帧则根据关键帧来绘制曲线；选逐帧将根据逐帧信息创建曲线。

曲线角度 ：曲线光滑度，决定用至少几个顶点来创建一个曲线。仅支持逐帧模式。

 创建路径动画 : 勾上创建路径动画的话，绘制曲线后将自动建立路径动画；

描绘按钮：根据上面选项执行相应动作。
