场景约束管理节点：

 

 

它只是一个简单的有nurbs curve形状的变形节点，通常跟某个视图摄像机绑定，在摄像机视图上稳定不动，方便切换一些属性。

这个节点被生成的目的，主要是为了可以统一管理场景里所有东一个西一个的约束切换权重属性及其动画 。

不过实际上你可以将任何数值类型的属性，收集在这个节点上。

 

访问方式：

在MGTools主工具条的约束工具图标上右键，可以访问到。

在约束工具条的大多数约束功能图标上右键，也可以访问到。

 

用法流程：

1.选择相应的要收集属性的节点;

2.在MG通道栏或Maya通道栏上，选择一个或多个属性;

3.用上面的访问方式，来收集到约束管理节点。将会弹出对话框，请您输入要连接到的新属性名称。

4.生成的约束管理节点将会被固定当前激活的视图上。样子如下：


 

提示：如果需要固定到其它视图，先激活其它视图，在同样访问方式用"固定约束管理节点到当前视图"来绑到该视图。

 如果不想要固定在视图上，在同样访问方式用"解除约束管理节点视图固定"来让它解除绑定。

 如果原先被收集的属性在收集前有动画，动画会被转移，原先属性会被约束管理节点上的对应属性连接控制。

 可以使用约束管理节点上的Icon size来决定图中那个感叹号图标的大小。
