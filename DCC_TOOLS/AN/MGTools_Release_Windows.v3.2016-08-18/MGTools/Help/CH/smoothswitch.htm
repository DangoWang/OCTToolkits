模型光滑快速切换功能：

   


用来帮助动画师提高切换场景里polygon光滑数值的效率。

相对智能地，如果polySmooth的divisions属性是被其它属性控制的话，它会去切换那个控制属性的值。

 

启动方法：

▪点击在工具架上点图标（如果你有装全MGTools工具shelf）；

▪在MGTools工具条上右击并向左划，在右键菜单选择MG模型smooth切换..

 

MG模型smooth切换界面：

 

第一栏筛选模式栏 ：决定是显示场景里所有的polySmooth项还是只显示Reference的，且路径含有/不含某字符的项。

                 后者主要是用来只显示角色/道具/场景中某一种的Reference的光滑项。

         所有项：显示场景里所有的polySmooth项。

         只限于：只显示Reference的，且路径含有某字符的光滑项。

         忽略：只显示Reference的，且路径不含有某字符的光滑项。

         *后两个模式选任一种，第二行的输入框就会激活，也可以点第二行最右的按钮快速输入一个常用字符。

          提示：切换模式时，下面的光滑项列表会自动更新。

全选中/全不选/反选按钮：用来操作下面光滑项的选择与否。

刷新按钮：若打开此界面后场景polySmooth有变动，点此刷新。

光滑项目列表：显示探测到的场景polySmooth项。注意如果polySmooth的divisions属性被其它物体所控制，则它会比较智能地抓到控制物体。

                 图中，每一个光滑项都分三个部分 ，第一个是复选框，勾上后，按画面右下角的"切换"按钮才会进行该项的切换；

                 第二个是当前数值，告诉你当前这个光滑属性的数值。第三个是针对该光滑项的单独切换按钮。

          提示：直接点击每项按钮，即是切换该项的smooth状况。若要选择该节点，则右键单击/选择该节点(如上图)。

清除Smooth控制属性上的帧：清除Smooth控制属性上的帧。要不然切换帧可能会导致光滑状况有动画哦。

互换值：用来切换的数值范围，如果是0-3这样的范围，切换时会按0->1->2->0->1->2...这样的顺序来切换。

切换!! 按钮 : 按列表里勾选的状况，对勾选上的光滑项执行切换。

  在有的场景里，对一个Polysmooth的divisions属性的控制可能会很复杂，如果是驱动关键帧，则该工具会继续找驱动的属性。但要是表达式控制的，则该工具会忽略。有时控制属性有最大最小范围，在切换时如果发现有大小范围的，会尽量在大小范围内切换。
