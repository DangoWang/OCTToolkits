配置Pose/动作库工具：

▪点工具菜单上的配置菜单 / 配置..

▪点工具菜单上的库工程菜单 / 管理库工程..

 

库工程管理：
这一tab，主要是用来管理库工程全局目录设置。
左边列出了所有本地的库工程。 列表上方有新建库工程按钮及重命名库工程的按钮。

右边，可以设置当前在左方列表选中的库工程的全局目录设置。

具体可以参考新建库工程的文档。

提示：编辑完后，要记得点"保存库工程设置"按钮，来将这些更动实际保存，同时保存后，如果该工程在主界面处于正打开的状况，会重新打开来刷新。
         也可以点重置，撤消界面上的目录编辑。

 

库工程管理：

这个tab里您可以设置生成缩略图及缩略视频的一些选项。
       

拍屏外观：

▪自动去除选择高亮：在生成预览图或预览视频前，自动把选择高亮先去掉。

▪自动隐藏Nurbs曲线：Nurbs曲线一般不用出现在预览图和预览视频里，默认在生成前会自动不显示。

下方则可以重命名，删除用户生成的摄像机书签。

 

杂项：

项目显示大小：指定大中小三个尺寸，最大不能超过320px,最小不能小于32px.

                 这三个尺寸，即是在主界面上点击放大镜按钮，来回切换的三个尺寸。

                 如果想显示大于320，最好在主界面上双击项的按钮，打开预览区域来预览。

确认： 决定再遇到特定情况前，是否要弹出对话框确认，给用户取消操作的机会。

▪删除项前确认：在主界面删除pose或动作前，是否确认。  

▪编辑项前确认： 在编辑项时，真正应用编辑前是否确认。

▪上传下载项目遇到已存在则确认：在上传下载数据前，如果目标已存在，提示是否覆盖等。

提示：基本上这些确认对话框的左下角都有一个"不再提醒"的复选框，勾上后就不再确认。如果想让它重新要确认，就在这边打开。

 

反馈 & 纠错： 决定主界面上的状态栏及maya脚本编辑器，在各种库操作过程中反馈打印信息的频繁度。
     太频繁会影响操作运算效率。建议选适中或沉默，如果有出现软件bug,可以开为罗嗦，以便反馈bug.

外部：决定图像浏览器及视频播放器的配置。
     目前请保持默认，即看图还是用pose动作库默认的控件，看视频则一定要用外部的操作系统默认的视频播放器。

提示：上面配置界面，除了库工程管理tab里的库工程目录设置需要另点按钮保存外，其它选项都是被改变时就保存。
