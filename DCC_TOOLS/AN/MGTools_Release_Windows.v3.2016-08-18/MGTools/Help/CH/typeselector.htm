类型选择器按钮功能：

 

对场景中的物体作一定条件的筛选并选择。

类似于Max按H弹出的按名称选择的功能。

注意，它只针对场景内所有的DAG节点物体。非DAG节点不在筛选范围内。

 

存在理由：

便于选择场景里所有的具有一定共性的DAG节点物体。

如，选择场景中所有有key帧的物体，选择某Rig 下的所有物体，选择所有的模型，所有的曲线等等。

 

类型选择器界面:

A区条件按钮：每个按钮点击后，都将在场景选择所有符合该按钮所标示的条件的节点。

只有勾选上该按钮左边的复选框，在界面最下方的选择按钮在执行时，才会考虑到该条件。

是/否被Key帧物体: 只能在两者间勾选一项或全不选。筛选的条件是该节点是否有被Key帧。

是/否Reference物体: 只能在两者间勾选一项或全不选。筛选的条件是该物体是否是Reference物体。

是/否被隐藏物体: 只能在两者间勾选一项或全不选。筛选的条件是该节点是否被隐藏。(即visibility属性为0)

是/否是被约束物体: 只能在两者间勾选一项或全不选。筛选的条件是该节点是否被其它节点约束。

有/否是场景所选Rig物体: 只能在两者间勾选一项或全不选。筛选的条件是该节点是否和当前场景所选的节点同处于一个最高层级下。

有/否是所选物体: 只能在两者间勾选一项或全不选。筛选的条件是该节点在场景是否被选择。

类型过滤: 只有勾选后才会出现B区类型列表。用来在满足上面勾选条件的同时，通过Maya的类别定义再做筛选。

全选/去选/反选: 将B区的列表项全选/全不选/反选。

B区节点类型按钮：每个按钮点击后，都将在场景选择所有属于该类型的节点。只有勾选上，在界面最下方的选择按钮在执行时，才会考虑到该条件。

自定义 : 勾上后旁边的输入框才会有效。在输入框输入自定义的类别名称用做筛选。

<- 获取 : 获取当前场景选择的第一个物体的类型名称并在左边的输入框输入。方便用户在不知道节点类型名称拼写的情况下使用。

下面的类型按钮 : (里面的(T)代表其要找的是该类型对应的变形节点(transform)，(S)找的是该类型节点，一般是形节点(shape)。)

▪Transform: 变形节点

▪ImagePlane: ImagePlane节点

▪All Geometry(T): 所有的几何体类型的节点所对应的变形节点。包含多边形，细分模型及nurbs所对应的变形节点。

▪All Geometry(S): 所有的几何体类型的节点。包含多边形，细分模型及nurbs。

▪...(其它具体见Maya帮助，应该按字面就能了解意思)

选择: 按整个界面里勾选上的条件，A区条件之前是"且"的关系，B区的类型条件是"或"的关系，来在场景中选择出符合条件的节点。

关闭 : 关掉类型选择器。

  注意，这个类型选择器的筛选对象只包括DAG节点物体，非DAG节点不会被选择。（非DAG节点，就是在outliner大纲视图里，去掉DAG Objects Only 勾选后新显示出来的节点。）
