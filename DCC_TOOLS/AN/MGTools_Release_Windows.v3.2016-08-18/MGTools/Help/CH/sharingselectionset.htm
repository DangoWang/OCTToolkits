共享选择集的流程：

1.由动画Leader制作出一套的选择集。

2.动画Leader在服务器上开辟一个目录，这个目录所有动画师都可以读取，但不需要可写。将建好的选择集手动复制共享到这个目录下。

3.动画师在选择集里用选择集路径管理器里，添加这个目录源。这样他们就可以使用这些集了。
 

选择集路径管理器启动方法：

▪在选择集工具上，左键点击案子旁边的栏位/选择集路径管理器..

▪在选择集工具上，左键点击>按钮/选择集路径管理器..

▪在选择集工具 杂项菜单/选择集路径管理器..

 

选择集路径管理器：

用来添加服务器上的选择集源。比如你的动画师leader，制作了一些目前项目的一些选择集，并共享在服务器上某一目录。
你可用这个管理界面，添加这个目录，这样你就可以在选择集工具里切换显示并使用这个目录下的选择集。

 

删除一个选择集目录源：这个只是从界面上删除这个目录显示，不会实际删除目录。

添加一个选择目录源：拾取一个目录，以添加一个选择目录源。

每一项目录源左边的复选框：用来作标记，选上后，如果点删除按钮/删除勾选 则只会删除勾选中的目录源。

路径短名称： 输入一个短名称，用来切换时显示在界面上。

路径：实际的选择集目录源路径。

允许编辑： 如果不允许，则选择集工具在切到这个目录源时，所有涉及到选择集生成或编辑的按钮将失效。

关闭按钮：关闭该对话框而不保存选择集路径配置。

保存按钮：关闭该对话框并且保存选择集路径配置，同时刷新选择集界面。
