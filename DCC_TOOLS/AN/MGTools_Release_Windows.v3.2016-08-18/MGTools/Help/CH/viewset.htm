视图配置切换按钮功能：

直接点击切换用户设定的两/三个视图显示切换方案。这一区功能主要是为了提高动画师设定场景视图的显示配置效率，另外附带的显示层集功能主要为了提高动画师建显示层的效率。

 

右键菜单功能：

编辑视图切换配置.. ：打开视图集配置窗口，进行2/3个视图显示方案的配置；

MG摄像机切换.. ：快速切换当前视图的摄像机。

MG模型smooth切换.. ：快速切换当前场景所有的多边形光滑。

无跳属性切换.. ：经过一些设置，无跳动切换一个属性，用在跟不跟随的切换时很好用。

新建显示层集.. ：把场景中所选物件新建为显示层集；

测试所有显示层集..: 把之前建立的显示层集都测试一遍，如果场景存在显示集中成员物件，

则自动建立相应显示层，并把物件放于这一层。

显示层集工具.. : 打开显示层集管理器，管理这些显示层集。

隐藏/显示所有显示层：隐藏/显示所有显示层。

藏入新层：把场景所选物体加入一个新层，新层将以"物件名_Layer"命名。

层编辑器..：打开Maya浮动显示层编辑器。

帮助..：打开MG视图集/显示层集帮助文档。
