动画缩放/帧率转换功能：

用来方便地缩放场景中的动画。可以同时缩放场景里的所有动画/所选物体的动画。

可以根据帧率不同来执行作为速度补偿的动画缩放。在缩放后还可以帮你自动吸附所有非整数点的帧。

 

典型用法：

1.打开动画帧 缩放界面如下图。

 

2.在缩放轴心栏里填入用来作缩放轴心的帧数。

3.在倍数里填入要缩放的倍数。

4.选择要缩放的对象模式和时间范围模式。

5.如果要在缩放后仍要保证全部都是整数帧，勾上最后一个选项。

6.点"缩放!!" 按钮进行动画缩放。

选项解释：        

帧率转换 ：如果你自己知道要缩放的倍数，则无须用到这一栏。这一栏主要是为了帮您计算出适当的缩放倍数而已。

      源帧率：源帧率设定,一般情况下填入当前场景中的帧率设定值，点获取按钮来获取当前场景中的帧率设定值。

      目标帧率：目标帧率设定。设定为你即将将设定的帧率，源帧率/目标标帧率都设定完后，即可点计算按钮来自动设定缩放倍数。

缩放轴心 ：填入用来做缩放轴心的帧位置。

缩放倍数 ：填入用来做缩放轴心的倍数。如果你是要转帧率又不想让已有动画速度有变，可以用帧率转换栏里的设定来自动算出倍数。

缩放的对象模式 ：

         所有Key帧物体 ：场景里所有的动画都将得到缩放，在帧率转换时常用。操作需要一定时间。

         所选Rig下的所有物体 ：当前所选物体所在的最高层级下的所有物体的动画都将得到缩放。

         所选物体 ：当前所选物体的动画得到缩放。

缩放的时间模式 ：

         全部：时间上没有限定，所有时间上的动画都将缩放。

         所选时间范围：只有在选定时间范围内的动画才得到缩放。

获取轴心 ：将当前帧的帧数填入缩放轴心输入框内。

缩放后自动吸附到整数帧 ：缩放完后自动强力吸附小数帧到整数帧。

缩放!!按钮 ：执行缩放。
