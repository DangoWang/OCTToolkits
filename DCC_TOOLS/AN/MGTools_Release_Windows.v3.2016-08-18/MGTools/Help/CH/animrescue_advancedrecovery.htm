高级恢复向导：

 

右键点击迷你工具箱图标，选择"高级恢复向导.."。或者选择自动恢复向导，在第二步时选择高级恢复模式也可以弹出高级恢复向导。

它不同于自动恢复向导，它可以显示出目前自动判断出的恢复点，你可以手动添加恢复点，也可以决定不使用某些恢复点。

目前是高级恢复向导支持从Maya 8.5~2012的x86版，Maya 8.0~2012的x64版本。以后更新的版本将通过在线更新来提供支持。

 

高级恢复向导界面:


 
点击"替换"按钮来拾取新的恢复点根目录从而重新评估适当的恢复点。旧的已认定适用的恢复点将被冲掉。

点击"添加"按钮拾取新的恢复点所在根目录从而扩大恢复点搜索范围。新的认定适用的恢复点将被加到旧的已认定适用的恢复点列表中。

  提示：这里的备份根目录设置与实际的备份目录无关，它只是用来暂时搜索恢复点用的。不会实际改变实时备份的根目录设置。

 

 步骤1：设定搜索条件 ：设定用来搜索的名字跟日期。恢复点所在文件夹必须以此名字为开头，并且修改日期要晚于该日期，否则排除；

点击各自右边的“<-” 按钮来快速获得当前场景的名称跟日期。如果当前场景是未保存的Untitled场景，则该功能无效。因为恢复这样的场景无意义。但是如果你需要，还是可以通过手动输入来得到恢复点。

步骤2：恢复点筛选：只有打上勾的恢复点才会被使用。恢复点每项都标注了场景名称，第几次保存，保存的时间日期等信息。

你可以点击全选，去选，反选按钮来快速选择，点击按钮来拾取一个目录来手动添加备份点。这个目录要么是备份根目录下那些以场景名称开头的目录，要么是这些目录下的各个细分保存目录。

点击下一步，再点"恢复"按钮即开始恢复。
