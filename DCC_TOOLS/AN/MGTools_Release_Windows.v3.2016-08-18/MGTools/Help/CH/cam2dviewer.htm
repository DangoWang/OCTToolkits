摄像机2D视图工具功能：

 

直接点击则将当前工具设为摄像机2D视图工具。

用来以2D试为放大/缩小/平移/旋转当前3D视图。

 

 

启动方法：

▪右击MGTools工具条的按钮。

▪直接点工具架上的图标(如果你有装全MGTools的Shelf)。

▪点shelf栏的按钮来快速恢复由造成的摄像机变动。
 

工具原理：

该工具实际上是通过一定算法来实时变动该视图摄像机的几个2D属性，如post scale,film translate, file Roll等属性来达到2D视图的操作感觉。
它可以在不改变焦距远近情况下放大缩小视图。在QC查穿帮流程中尤其有用，动画师亦可以用它来放大局部来调细节。

在Maya2011中，该工具鼠标与之前版本Maya的工具鼠标显示效果将不一样，用的是内置鼠标。(maya2011。。。你实在特殊到另人发指)

 

使用方法：

▪工具被激活后，在视图上直接用鼠标左键拖拽来以2D方式放大/缩小当前视图。

▪直接按鼠标中键拖拽来上下左右平移视图。

▪按Shelf+左键拖拽来以视图中心为中心旋转视图。默认选项下，将以45度倍数进行吸附。

▪按Shelf+左键点击视图来恢复旋转为零。

▪按Ctrl+左键来拖拽一个框来放大/缩小某区域。

▪按Ctrl+中键点击视图来恢复所有视图改动。也可以按键盘的回车键。

工具选项：双击Maya工具栏的当前工具图标，来打开工具选项窗口。或者在MGTools按钮上点右键/工具选项..。

 

▪保存前自动撤消所有视图操作： 当Maya里执行保存/另存前，是否自动撤消所有由摄像机2D视图工具引起的摄像机变更。
这一选项实际上是用来决定这种视图操作结果是否保存下来。
可以用来防止在不知情情况下不小心变动了摄像机而不自觉。

▪自动吸附旋转到45度： 决定是否自动吸附到45度旋转。
 

▪立即撤消按钮：点击立即撤消场景里受摄像机2D视图工具影响的摄像机变动。

         你也可以点shelf栏的按钮来快速恢复。在MGTools的按钮上点右键/撤消所有视图操作也可以。
