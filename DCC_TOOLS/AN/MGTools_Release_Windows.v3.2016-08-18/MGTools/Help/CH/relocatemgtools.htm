如果您的机子原先MGTools Pro3是以本地模式安装，现在想转为服务器模式，或者想在服务器模式及完全服务器模式之间转换，或者原先就是以服务器模式安装的MGTools, 但是服务器端的MGTools源被移动，删除，不可用，现在需要指定另一个服务器上的目录为MGTools源，则会用到MGTools Relocation工具。
请采用下面步骤：

1.点击MGTools这个工具架上的第一图标，即图标。打开MGTools Relocator。
提示： 如果是服务器的MGTools源被移动，则此时MGTools工具架上应该只剩下这个图标，其它图标全部挂掉。

2.在弹出的MGTools3 Relocation窗口中占击红色的"Pick MGToolsLoader"按钮(该窗口不支持多语言，永远是英文界面)：

 

3.如果想在服务器模式及完全服务器模式之间转换，请开关“Use full server mode (Copy nothing)”选项。

4.弹出拾取对话框中，拾取到新的服务器端MGTools文件夹中的MGToolsLoader_WrittenByMiguel.mel文件

5.这个工具将设置MGTools指向目标文件所在的新目录。过后MGTools应该会自动弹出。

6.右键点击MGTools第一个图标 / 安装/卸载 / MGTools工具安装器...，使用工具安装器，将失掉的MGtools工具架工具重新安装回来。
