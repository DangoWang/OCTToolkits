批量生成预览图：

在主界面上勾选一些要批量生成预览图的资产项，然后下面方式任选一种来打开批量生成预览图对话框:

▪点击主界面的按钮。

▪主界面资产项菜单/ 批量生成预览图..

        注意: 因为涉及到不确认情况下批量开档，所以如果您当前场景处于未保存状态，会先提醒您是否保存档案。如果不是点取消，都将随后弹出批量生成预览图对话框。

 

批量生成预览图对话框：        

界面左侧列出了之前勾选上的资产项.

每个资产项左边的图标，标示了目前状态。时钟图标代表等待生成。绿色箭头代表当前正在生成，红色感叹号代表出错，绿色勾代表成功生成。

右边整个区域： 是MG-Pose动作库及MG资产库通用的生成预览图，预览视频的界面。
下方的拍屏之前选项：

▪停住这样我可以选择一个视图摄像机书签：每次开档，都会停下来，以便用户调整捕捉视图的摄像机位置等，然后再继续。

▪使用默认视图摄像机书签： 每次开档，捕捉视图会自动应用~[4]default这个摄像机书签然后进行生成。预览图生成过程不会停歇直到全部生成。

▪使用视图摄像机书签：选定后右边出现书签选择下拉菜单。
每次开档，捕捉视图会自动应用您预先选中的摄像机书签然后进行生成。预览图生成过程不会停歇直到全部生成。

文件模式：决定了预览图遇到已存在时的覆盖模式。

▪追加到既有预览图：会自动避免命名与即有的重复，不会覆盖即有的预览图。

▪替换即有预览图：生成前先清空预览图数据，再生成。  

点击生成!!按钮，开始批量开档->生成预览图的过程。

关闭按钮： 关闭该对话框。

    注意，这种方式生成的预览图，其图片一定是存在于:
         当前Maya用户目录/MGTools_GlobalData/MG_AssetLib/资产库工程名称/Preview_Images下面。

         在这个目录下面的图片，是通过相对路径的方式，跟相应的资产项绑定。

         所以在上传下载中，可以同时上传Preview_Images下面的图片到全局库的Preview_Images下面,就能访问到这些图片。
