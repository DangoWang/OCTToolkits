MB-保存器：

 

 

这个工具用来提升您保存Maya场景的效率，比如自动新增版本号，

保存前视图清理等等。

 

访问方法：

▪这个功能直接点击Maya工具架上的保存前处理图标：及保存图标: 比较方便。

▪点击图标打开迷你工具箱，点击"MG-保存前预处理“按钮及"MG-保存器"按钮。

▪右击图标，右键里也有同样功能。

▪最方便的方式，是直接在MG配置窗口里，将这两个功能或一个功能设置快捷键。用键盘访问最快。

 

MG保存器的选项：

要设置MG保存器的相关选项：

▪打开MG配置窗口。找到工具配置这一栏，在下面工具列表找到MG-保存器，在右方区域设置。

▪右键点击迷你工具箱图标，在MG保存器或MG-保存前预处理这两个菜单左边点击其选项框。

 

选项界面：

保存前预处理选项：设置在保存前，要做哪些处理。

去掉任何物体显示：将视图的所有类型对象都去除显示。

显示模式：可以选择线框，Bounding Box等, 也是决定视图的显示方式。

所有的显示层： 可以将所有显示层关闭或打开。

设置MasterLayer为当前的渲染层：将Master Layer设为当前渲染层，这样灯光阶段可能比较方便。

前往帧数： 保存前将当前帧跳往某一帧，一般设置为第一帧。

用户脚本： 上面的预处理方式不可能照顾到全部需求，您可以在下面写入您自己的脚本，

并可以设置这些脚本排在其它处理项之前还是之后处理。脚本支持Mel或Python, 可以点测试按钮测试用户脚本的效果。

安装到工具架： 将上面设置的预处理功能，安装到当前工具架，以后点击这个工具架即进行预处理。

设置快捷键： 为预处理功能及保存功能设置快捷键。

版本添加选项：设置在保存前自动加场景版本号的规则。

分隔符：决定版本号跟场景名之前的分隔符。请根据您制作的项目的命名规则来设置。

倾向的数字位数：设置版本号的位数，设置3位则会加入001这样的版本号。

当前结果：显示出当前场景，按上面设置的保存前自动加场景版本号的规则加版本号后的结果。

Maya场景保存选项：点击打开Maya保存场景的选项窗口。

保存场景：点击则按上面设置的加版本号规则，来保存您的场景。

 

修正学生版Maya保存Bug：

在迷你工具箱的MG-场景保存这一栏下面，还有这样一个按钮：

这个功能用来修正有时Maya学生版，无法保存场景的Bug.

这是Maya本身的Bug, 不过以前MG动画银行则时常触发这个Bug，目前MG动画银行工具已经修正 。

但很可能您安装使用的其它工具同样会触发这个bug, 所以这里同时提供这个按钮，如果您是用的学生版的Maya，则这个按钮可用。点击后，不可保存的按钮即可以保存了。
