MG重定位器功能：

生成一个控制装置用来重新摆放要重定位的多个物体，

重摆导致的位置改变将融入到这些物体原有的关键帧中。

* 这个工具启发于Indigo大人的relocator工具，因为原Layout同事要求，所以重写代码并加入。有作一些改进，如支持重定位器的动画等等(这个支持相当于多了一个暂用的动画层! )。

 

典型用法：

1.打开重定位工具界面如下图。

 

2.在场景里选择你想要重定位的物体。当然可以选多个。点上图中的+号按钮加入到下面的重定位列表中。
（如果在开启界面前有选物体则所选物体将自动被加入列表）

3.点击上图中左下方"开始重定位!"按钮（该按钮会马上变成"完成重定位!"按钮，同时旁边的"选择重定位器"按钮被激活）。
场景中新生成并选择的空组即是重定位控制器，旋转，位移这个控制器以重新定位这些物体。
也可以Key动画，key的动画将融入重定位的物体原有的关键帧里。

4.点"完成重定位"按钮。做的所有改动将反应到被重定位的所有物体的关键帧中。

提示：在重定位过程中，因为用来重定位的层级只是个空层，如果想重新选中重定位器，可以按"选择重定位器"按钮来重新选择以便接着旋转，位移。
