MG通道栏按钮功能：

 

直接点击即打开MG浮动通道栏。

MG通道栏有Maya通道栏全部功能，亦有自己特色的功能。

 

MG通道栏特色功能：

▪界面可以浮动也可以固定在屏幕不可动。

▪整合其它编辑器如Outliner, MG选择集，层编辑器，曲线编辑器等进界面，也可以决定它们是内置还是浮动。

▪将动画最常用功能化为小图标，置于界面头部。

▪快速重置MG通道栏上所选通道为默认值。如旋转位移归零，缩放归一，自定属性归为各自默认值。

▪在曲线编辑器中只显示在MG通道栏上所选通道的动画曲线。

▪快速bake在MG通道栏上所选通道的动画。

▪跨Maya间拷贝数值的功能。

▪非常丰富快捷的markingMenu功能。

▪2.0版的在右键内新内置了一些功能。

▪2.0版的MG通道栏取消了建立scriptedPanel的机制，这样在Maya视图菜单的Panel菜单里不会再出现像MG_channelBox这样的panel了，
这样做是为了使这个工具更加稳定。

▪2.5版的MG通道栏在Maya2011及其以后的版本，支持dock停靠在Maya主界面的左/右侧，就像Maya通道栏那样。并且可以在MG通道栏选项窗口里选择是停靠还是浮动，默认是浮动。
 

右键菜单功能：

MG通道栏按钮的右键采用的是MarkingMenu的方式，这样在操作时只需要鼠标快速划过即可完成操作。

 

重置所选属性 ：重置Maya通道栏所选通道。

隐藏/显示所有显示层：显示/隐藏所有显示层。

藏入新层：把场景所选物体放到一个新建的层里并隐藏。该层自动以物体名称+"_layer"方式命名。

层编辑器..：打开浮动的层编辑面板。

设置工程目录..：设置当前工程目录。

新建工程..：新建工程目录。

曲线编辑器插件开关：开关曲线编辑器插件显示。

自动启动该插件：决定曲线编辑器插件是否自动随Maya曲线编辑器启动而启动。

MG通道栏选项..：打开MG浮动通道栏设置窗口，决定启动MG通道栏时是否最小化MGTools。
                            因为后者常常被前者挡住。还可以决定是否显示MG通道栏的菜单栏。

帮助..：查看MG通道栏帮助。

 

MG浮动通道栏界面：

菜单栏: 按对象归类。第一个是动画/Key帧的操作，第二是属性的操作，第三是打开各种编辑器。最后的一项属于杂项。

 图标区：动画常用功能图标区。

依次是：key帧所选通道，删掉所选通道帧，重置通道值，位移归零，旋转归零，Mute所选通道，

               同步曲线编辑器通道选择，在时间滑条只显示所选通道，及鼠标中键灵敏度。

在B区这一栏中内置了三套MarkingMenu右键菜单:

Shift+右键：将弹出MG吸附定脚工具相关的MarkingMenu。

Ctrl+右键：将弹出动画常用的编辑器的快速MarkingMenu.

Alt+右键：将弹出属性操作的快速MarkingMenu。如加,锁属性等等。

提示：在图标上直接点右键，将会出现各自相关的功能，因为比较多且动画师都比较熟悉，在这就简单说明下面一项的右键：
     

在曲线编辑器里只显示所选通道：在曲线编辑器里进行隔离显示通道，只有在MG通道栏中选中的属性才显示/选择。

曲线编辑器插件开关：添加/删除曲线编辑器上的GEplugin按钮，详细下章。

通道栏本身： 右键做了重新组合，把极常用的放在MarkingMenu头部，在这里简单说明几项：

MG动作银行.. : 启动MG动作银行，你可以用它来导出导入动画。

动画帧探测器.. : 启动动画帧探测器，来查找出，哪些对象在驱动当前角色的动态。

拷贝/粘贴数值.. : 在Maya间拷贝粘贴数值。如果你拷贝时有选通道属性则只拷贝所选属性，如果没选，就会粘贴所有在通道栏有展开的属性(包括                                 所有展开的所选节点/shape节点/ 输入输出节点的属性)。在导入的时候，只要确保想要粘贴的属性在MG通道栏有展开显示出来就             好。只要有拷贝过数值，不管什么时候再打开Maya都可以粘贴。
在C区除了直接点右键的MarkingMenu，还另外内置了三套MarkingMenu右键菜单:

Shift+右键：将弹出选择集操作相关的MarkingMenu。更重要的是，它的下部就是选择集的内容，你在选择集的任 何操作就实时更新到这里。最底下的更新菜单是用来手动更新这个菜单的。所以你制作的选择集在                   这里就可以快速用到~！！

                    在菜单的下部分就显示了当前场景的所有显示层，选择中它们将显示/隐藏它，点选项就是编辑该层的

               属性。在Template/Reference/删除层/创建/加到层子菜单里选择中相应的层即是对该层做相应的操作。
               如果你的层一向命名有方，用它就像用层编辑器一样，只是不用再开一个面板来占用工作空间。
