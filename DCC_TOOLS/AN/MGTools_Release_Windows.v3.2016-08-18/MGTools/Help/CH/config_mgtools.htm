MGTools配置按钮功能：

 

MGTools配置。 在此进行MGTools的一些配置。

比如主要功能的快捷键 ，Maya启动自加载设置，MGTools工具安装。

 

启动方法：

▪在MGTools工具条的第一个图标点右键/MGTools 配置..，即可弹出此配置界面。

▪直接点工具架上的图标(如果你有装全MGTools的Shelf)。
 

MGTools配置窗口：

 

  MGTools界面设置：


 

MGTools语言：切换MGTools的全部界面语言，目前支持英文，简体中文及繁体中文。

                       MGTools的工具界面需要重开来刷新语言；

MGTools图标：切换MGTools的图标，2016及以后的maya版本默认是MGToolsPro3.1新增扁平化彩色图标，2011~2015默认用的是MGToolsPro3新增的彩色图标。

                       MGTools的工具界面需要重开来刷新图标显示；

MGTools界面: 用来筛选哪些图标不要留在MGTools主界面条上。

           在这一栏被选中的工具，则是在主界面上将被隐藏的。

           点击“保存界面配置”按钮，则将这个选择实际应用并保存。

           这些界面选择的配置保存在MGTools_LocalData/MG_UIConfig/MG_ToolsChoice.cfg里。

 

MGTools快捷键：设置MGTools相关功能的快捷键。

A区 ：显示所有提供的可被设置快捷键的MGtools功能名称；这个要比Pro2.0时多出很多的可设置的项。

     基本上所有主要的MGTools功能都可以被设置快捷键了。

     可以在下方的快速搜索栏输入关键词，来过滤显示列表，以便快速找到要设置快捷键的功能项。

B区 ：在这个区设定所选功能的快捷键；如果所选功能已设定过快捷键，则该区将自动显示其快捷键。

C区 ：显示当前A区列表所选功能的实际Mel代码。

提示：在A区列表上点右键：

 

    移除当前项快捷键: 删除当前列表所选项目的快捷键；或者直接选择列表项目，按键盘上的Del键，即可删除选定项目的快捷键:P

    移除所有MGTools相关的快捷键: 删除当前列表所有项目的快捷键；

 

 

MGTools自动加载项配置：


配置保存在： 决定MGTools随Maya启动，要启动哪些工具的配置代码，要写在Maya用户脚本文件夹下的userSetup.mel还是userSetup.py. 有些生产环境下，userSetup.mel被禁用而无效，这里可以选择userSetup.py.

在选框区中选中想要自动启动的项目,选中MGTools菜单并点保存自动加载配置按钮的话，如果当前Maya菜单栏中没有MGTools菜单将自动安装。

安装到工具架及菜单栏：

在选框区中选中想要安装的项目然后点击 " 安装/更新 "按钮来安装/更新。
安装全部到当前工具架：全部MGTools主要功能都安装到当前工具架。
安装全部到 "MGTools" 工具架：全部MGTools主要功能都安装到"MGTools "工具架(没有则新建)。
点按钮，将弹出MGTools 工具安装器窗口。

提示：在" 安装/更新 " / 工具架安装器 按钮上点右键：

删除当前工具架所有MGTools Shelf : 在当前的工具架删除所有的MGTools Shelf。

工具配置：    基本上大多数MGTools工具，有涉及到选项的，都会集中到这里。不过有些大的工具，还是有自己的选项窗口。

   选择列表中不同的工具项，来设置相应的选项。
