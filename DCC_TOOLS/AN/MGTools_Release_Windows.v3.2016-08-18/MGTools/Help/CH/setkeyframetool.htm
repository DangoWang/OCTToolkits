Key帧工具功能：

用来按一定间距快速设Key或Key帧每个关键帧。
2.5版的解决了会将关键帧数值全部抹平的重大bug。
 

典型用法：

1.打开帧工具箱，点开Key帧工具界面如下图。

 

2.在场景里选择要设Key的物体。

3.选上图两个模式任何一种，点"设key"按钮来快速设Key。

界面选项的解释：        

每个关键帧 ：为所选物体在每个关键帧处都打上Key。其实就是确保每个关键帧处所有可设Key的属性都有设key。

每...帧 ：为所选物体在每隔一定数量的帧间隔打上Key。

          提示：选这个模式的话，要注意当前帧在哪个位置对K帧结果是有影响的，因为它会保证设的key会有一个落在当前帧。

设Key按钮：按上面两个模式中选定的一种模式设Key。        

          提示：不论选哪个模式来Key帧，如果你在时间滑条上有选定的时间范围，则只会在该项范围内设Key。        
"设Key"按钮上点右键 / 设Key非默认值的通道：将设Key所选物体所有可Key帧的，且处于非Rig默认值的属性通道。

"设Key"按钮上点右键 / 删除无动态的属性Key：调用Maya的"Delete Static Channels"功能。
