约束管理器按钮功能：

   

 

直接点击打开MG约束管理器。

或者可以直接点击工具架上的图标（如果有装全MGTools的Shelf）。

 

特色功能：

▪可以快速选择被约束物体的约束节点及约束物体，以便查清谁被谁约束。

▪快速地清除约束。

▪纠正刷新约束权重属性名。

▪可以随时把约束物体从约束列表里去除。

▪可以将列表内的约束导出导入约束偏移。

 

约束管理器界面：

选择好被约束体，然后点或者图标来打开约束管理器界面，下面介绍按钮及菜单：

约束节点列表 ：显示被约束体的约束节点，双击列表项则选择相应约束节点；

约束物体列表： 显示参与约束被约束体的物体节点，双击列表项则选择相应约束物体；

选择约束节点：作用同双击面的左列表，约束列表所选的约束节点。

修正权重属性名 ：纠正刷新在左列表所选约束节点的约束权重属性名。

删除约束：清除在左列表里选择的约束，或者直接按键盘Del键。

清除约束：清除所有约束。

导出约束相对偏移..：将左列表所选约束节点的约束相对位置信息导出成文件。

导入约束相对偏移..：将约束相对位置信息文件里的信息导入给左列表所选约束节点。

清除约束相对偏移文件..：在Maya的Temp暂存文件夹里清除所有约束相对位置信息文件。

从目标列表里移除：去掉右列表所选的约束体的控制，或直接按键盘的Del键。

修改约束轴向：打开窗口修改左列表中所选约束节点的约束轴向。

重选被约束体并刷新：重新选择被约束物体并刷新约束管理器。

刷新界面：重新载入场景所选物体作为被约束物体进行约束管理。

提示：在左/右列表上点右键亦可访问到相应功能。上面有些功能是这些右键才能访问到的。
