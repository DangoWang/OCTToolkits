MG选择集按钮功能：

 

直接点击打开MG选择集窗口。

这一栏主要是针对Maya的控制器选择方面。

 

特色功能：

▪以可以流程中快速共享你的选择集，同时可以在界面上使用别人的选择集存储目录。

▪以外部文件形式存储用户的选择集。

▪这些选择集在MG选择集窗口可以以长名字列表形式列出，也可以Shelf方式列出（如下图），同时后者支持自定义图标。

▪生成方便，直接选择控制器点生成按钮。

▪管理方便，可以随时更名，再编辑；编辑时支持当前场景选择的添加或删减。

▪可以做重定向。由一个角色转向另一角色相似控制器选择。

▪如果有不存在的情况，它会继续选择存在的控制器，选择完后会提醒你不存在的数目跟选择的数目。(Shelf模式除外)

▪支持多个选择集同时选择。

▪可以把选择集用locator备份于场景，随场景保存，也可以再调用出来。

▪选择集可以与Shelf互转，同时会更新MG通道栏的选择集内容MarkingMenu菜单，场景的选择集MarkingMenu.

▪支持在选择集目录下用文件夹分类选择集。推荐根据不同案子建不同的文件夹来组织。

▪所有选择集都可以用做在视图里做Isolate的集。

▪界面里一栏显示集，一栏显示集内的成员。直接在成员列表里选择相应项则直接在场景选择相应物体。

▪2.0版的选择集取消了建立scriptedPanel的机制，这样不会再新建叫MG_selectionSet这样的panel了，
这样做是为了使这个工具更加稳定。

▪2.5版的选择集在新建/编辑选择集对话框里增加了加入场景所选及去掉场景所选的按钮，原先这个功能在右键里。

 

MG选择集界面：

长命名列表模式与Shelf模式：

 

UI界面分为菜单栏，文件夹选择区，常用功能图标区，选择集列表/shelf区和成员列表区 。

菜单栏: 里的功能跟界面上的功能一一对应，这些功能在功能图标区上直接点击或点右键也可以访问到，或者在列表上点右键也可能访问。

案子：在这一栏直接点击，将弹出选择子目录的菜单（如上左图）。"*ROOT*"表示根目录，Project1,2表示根目录下的Project1,2文件夹（分类）。

           选择不同的目录，下面的内容将更新。
            在这一栏点右键，（如上右图）。创建新项目即新类别，刷新用来更新文件夹显示，也可以选择打开选择集文件夹来管理选择集。

图标区：

：点击在长名列表模式与shelf模式切换。

：按所选的选择集项目来进行选择。在Shelf模式则直接点击shelf.

：将所选选择集/选择集成员转向选择。基于reference流程，通过换reference namespace来实现。

：将所选选择集/成员作为isolate集，在当前激活视图中，只显示集内的成员，其它的隔离不显示。

：将场景所选物体存为选择集，将弹出新建窗口。

：编辑所选的选择集。

：快速重命名所选选择集。

提示：在俩图标上点右键会有相关功能，大部分功能都集中在右键上。所有命令说明如下：

：编辑所选的选择集转为选择用Shelf安装在Maya工具架。

：删除所选集（或按键盘上的del键）。

选择集列表区: 在选择集列表上点击选择一个列表项，在成员列表那边就显示出该集的所有成员。

双击列表项，则是应用该选择集，即在场景里选择该集的所有成员。

成员列表区: 在成员列表上点击选择列表项即在场景里选择该物体。支持多选。

这样用户不仅可以应用选择集，也可以应用其个别的成员来更灵活地选择。注意，双击成员列表项则是在场景里isolate该物体。

列表区上面也有两个按钮，分别对应isolate所选成员及重定向所选成员。

 

所有菜单命令：

菜单栏开关: 开关上面的菜单栏，这个功能在右键可以访问到，以节省屏幕空间。

选择管理器 : 开启选择管理器，来暂时管理前后选择或拷贝动画。

在左列表列出.. : 把选择集反映在选择管理器左列表。

在右列表列出.. : 把选择集反映在选择管理器右列表。

将所选选择集安装到工具架: 把所选选择集转为shelf安装在工具架上。

所有集安装到工具架: 把所有选择集都转为shelf安装在工具架上。

选择用shelf到选择集转换器 : 将启动转换器，在那里你可以拖已有的选择用shelf进转换器，转为MG选择集。

打开选择集文件夹..: 打开这些选择集文件存储的文件夹以便做异地备份等等。

将所选集随场景保存: 以Locator方式将所选选择集文件备份于当前场景。

刷新: 刷新选择集列表。

使用所选集: 根据选择集执行选择，相当于按钮。

重定向所选集: 重定向选择，会弹出对话框提示输入新的namespace以转换选择到新角色。

新建所选集 ..: 将场景所选物体存为选择集，将弹出新建窗口。功能等同于。

编辑所选集: 编辑所选选择集，功能等同于。

重命名所选集..: 快速重命名选择集，功能同。

删除所选集 .: 删除所选选择集项目，你也可以直接按键盘上的DEL键。

切换长列表/Shelf模式: 在长名列表模式与shelf模式切换。

帮助 .: 查看MG选择集帮助文档。
