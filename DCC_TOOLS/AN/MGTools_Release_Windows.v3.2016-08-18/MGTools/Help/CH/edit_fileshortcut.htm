编辑文件快捷方式按钮功能：

 

启动程序/文件快捷方式管理器来管理用户内置的程序/文件快捷方式。

 

启动方法：

▪在MGTools工具条的点右键/编辑自定义文件..，即可弹出此管理界面。

▪直接点工具架上的图标(如果你有装全MGTools的Shelf)。

 
文件快捷方式管理器界面：

默认将打开：显示目前直接点击MGTools上图标将启动的程序/文件。

A区列表 ：显示当前用户已内置的程序/文件列表。

第一个列表是菜单名字，第二个是实际文件路径，第三个是文件的注释。

提示：在列表上点右键：

编辑: 把所选项目置于B区的编辑状态。功能同如图中间的编辑按钮。

设为默认: 把所选项目设为点击MGTools上图标的默认启动项目，功能同如图中间的设为默认按钮。

上移: 把所选项目向上移一个位置，功能同如图中间的上移按钮。

下移: 把所选项目向下移一个位置，功能同如图中间的下移按钮。

删除: 删除所选项目。你也可以选好项目直接按键盘上的Del键删除。

重加载: 刷新列表的显示。

B区：文件快捷方式的编辑区。在A区选中项目按Edit按钮即可打开B区。

菜单名: 显示在右键菜单的名字。

文件路径：该文件的路径。可以按重拾取按钮重新拾取。

注释：该文件夹菜单命令的注释。

提交当前内容编辑：把当前在B区的编辑提交到A区的列表。

保存全部编辑：把所作的编辑真正存储。

关闭：退出文件夹快捷方式编辑器。

  在B区编辑完后，记得点提交当前内容编辑按钮来把结果提交给A区列表；

满意现在的编辑后，一定要记得按保存全部编辑 钮保存修改，否则将视为取消。
