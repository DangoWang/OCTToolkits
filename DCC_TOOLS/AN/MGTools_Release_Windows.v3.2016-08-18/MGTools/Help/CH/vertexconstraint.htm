顶点约束功能：

生成可以粘着于模型表面某一处的控制器，可以用这个控制器来约束别的物体。

有点类似于著名的Rivet脚本，不过它的好处是，即使模型光滑了也不会出错。

生成时支持的对象有： Polygon的点，线，面； nurbs的cv控制点。

 

用法：

 

1.选择Polygon的点，线，面或nurbs的cv控制点，Hull 这五种中的一种。

2.点击工具架的按钮（如果有装全MGTools的shelf），或者约束工具条上的相应图标。

3.这时即生成一个圆球型的控制器，它属于一个nurbs curve物体。可以用它来约束其它物体，使该物体粘着于模型表面。
随表面变化（如blendshape等）而变化。

  生成的控制器的位移是很稳定的，但是旋转在有的情况下不太稳定。针对这个问题，MGTools的更新已提供新型的UV约束工具，根据物体表面的UV坐标来做约束，它的旋转和位移都将很稳定。请优先使用那种约束吧 :)

  另外，生成的控制器的个数是由你选择的次级物体的个数来决定的。如果你选的是一个边，则生成一个控制器。如果你选的是两个顶点，就是每个顶点生成一个控制器。

提示：在约束工具条的上点右键/ 清除所选顶点约束器:  先选择顶点控制器，再用此来清除整个顶点约束装置。（只有这样才能清除干净，因为除了清除该装置层级，还要清除相应的set）
