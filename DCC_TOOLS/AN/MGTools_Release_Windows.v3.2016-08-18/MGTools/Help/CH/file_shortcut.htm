文件快捷方式按钮功能：

 

直接点击即打开用户设置为默认的程序/外部文件，如果用户未设置过，则弹出拾取外部程序/文件窗口。

这个按钮主要是处理外部程序/文件的快捷方式。

 

右键菜单功能：

计算器..以上区域 ：用户内置的外部程序/文件快捷方式将以菜单命令方式内置在这一区域的菜单中；

计算器.. ：打开Windows计算器来算帧数，工资，哈哈~~~

添加自定义文件..：添加一个用户外部程序/文件快捷方式。

编辑自定义文件..：编辑用户内置的程序/文件快捷方式。

移除默认行为..：把设为默认启动程序/文件的项目清空。（即直接点将启动的程序/文件）

 帮助..：查看快捷程序/文件的帮助。

 

添加用户快捷程序/文件：

在上点右键/添加自定义文件..即可开始添加用户快捷程序/文件。

点击后，会弹出拾取文件窗口,

拾取你想要的exe程序/任何文件后点确定，会弹出新建程序/文件快捷方式窗口：

第一栏：显示你之前所拾取的程序/文件目录。

要在右键菜单里显示的名字：将来内置在右键菜单里显示的名字，如果存成shelf就是shelf的标签。

工具架标签.：如果要存成shelf，这一栏的文字将作为shelf图标上显示的名字。

保存为默认：如果将此程序/文件的打开作为点击默认的功能。

保存到右键菜单按钮：将这个程序/文件快捷方式内置于MGTools的右键菜单中。

安装到工具架按钮：将这个程序/文件快捷方式作为shelf安装在工具架上。

关闭按钮：关闭这个对话窗口。
