相对吸附的启动方法：

▪右键点击在MGTools工具条的图标/ "向前/向后相对吸附"。

 

相对吸附的使用时机：

 

有时你想道具像父子约束在手中拿的那种效果。

可是道具可能会经过很多角色的手约束，所以会经历很多很多约束权重开关什么的很麻烦。

这时可以用这种逐帧模拟的效果来快速实现。无约束，只靠帧来模拟。当改变了手动作，只需要花一两秒重新模拟就可。

 

相对吸附的使用方法：

1.选择两个控制器，比如一个是手控制器，一个是道具控制器。

2.点击上述的图标/ "向前/向后相对吸附"，场景将在道具控制器轴心处产生一个暂用的Locator，并且时间向前跳一帧或后退一帧。

3.按键盘的“G“键，一直重复这一命令（也可以一直按图标)，同时观察视图上道具控制器的状态，它将被逐帧吸附到相应位置。

4.当动画到了道具可以离手的情况，停止按G键。

5.播放动画，道具就像被手约束一般。

  友情提醒：有时约束会比较省事，但是需要多次编辑到多个约束权重时，有的短期的约束还不如用帧来代替。

 

  提示：在Pro3.0版本中，选完要吸附的对象后，可以先在时间滑条上先选定一个时间范围，这样直接点一次吸附，则会吸附整个时间范围。
