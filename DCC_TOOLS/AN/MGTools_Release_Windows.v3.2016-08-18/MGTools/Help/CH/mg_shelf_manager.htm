MG工具架管理器按钮功能：

 

MGshelf管理器。

点击它将弹出MGshelf管理器窗口，可以进行用户Shelf的内置及编辑。

 

启动方式：

▪在MG工具架界面上点击按钮即可打开MGshelf管理器。

▪直接点工具架上的来启动(如果你有装全MGTools的Shelf)。

 

MG_Shelf管理器界面：

A区 ：已内置的用户Mel将出现在这一栏。

类别编辑区: 用这一区的按钮来上移/下移/新建/复制/改名/删除类别。

保存按钮: 保存这一栏的shelf.

取消编辑: 如果之前处于编辑某shelf状态，点此取消编辑，编辑区将清空并收起.

提示：在A区shelf图标上点右键，将出现菜单如左图：

编辑：将把该shelf放到下面B/C区的shelf编辑区里。

删除：直接把这个shelf删掉，和中键拖该区shelf到右边垃圾桶图标是一个效果。

重置图标为默认..：将该Shelf的图标及经过图标重置为MGshelf 的默认图标及默认鼠标经过图标。

B区：将显示上面编辑状态的shelf的实际命令内容。用户也可以自行在里面输出命令。

mel / python按钮：指明输入框中的命令是mel还是python。

测试代码按钮：测试执行B区输入框中的命令是否正确。

存为mel文件：把上面输入框的内容存为外部mel文件，以mel的source方式执行上面的命令。

提示：在B区输入框中点右键，将出现菜单如上图：

快速插入 source  mel 代码..：点击将弹出对话框，选取你想要source的mel文件，确定后在输入框里将出现source这个mel文件的语句。

快速插入 source  python 代码..：点击将弹出对话框，选取你想要import的python文件，确定后在输入框里将出现import这个python文件的语句块。

  根据当前编辑的是mel还是python，这两个菜单项会有一个是灰掉不可用的。

C区：所编辑/生成Shelf的标签，图标的编辑区。

第一个图标：点击选择一个32X32 位图作为shelf图标。

第二个shelf区：直接从Maya工具架上ctrl+中键拖一个图标下来作为shelf图标。

第三个图标区：点击选择一个32X32位图作为shelf鼠标经过图标。

  编辑完后，记得点最后的添加按钮来在A区生成shelf.(即使在编辑时)

满意现在的编辑后，一定要记得按保存按钮保存修改，否则将视为取消。
