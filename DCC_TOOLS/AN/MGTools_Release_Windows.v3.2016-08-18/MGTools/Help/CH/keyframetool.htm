MG帧工具箱按钮功能：

 


直接点击将打开MG帧工具箱。

这个工具的功能都围绕关键帧的快速生成及处理上。

 

这个工具的Timing工具，中间张工具，切线类型工具，启发自Lionel GALLAT的retiming tool。是应同事要求，加入MGTools的。

不过这里只是借鉴它的部分设定，代码重写后并加以改善，增加了一些选项。

在MGTools主界面的按钮上点右键，可以访问到该工具基本上全部功能，合在工具界面上介绍：

 

帧工具箱界面：

 

MG帧工具箱目前分为五栏，每一栏都可以点它的标题栏来开合，标题栏旁边的是帮助按钮，点击访问每一栏的帮助。

杂项工具 ：里面包含很多杂七杂八的跟关键帧处理有关的小工具。

Maya 间复制动画/Pose

MG动画记录器

重新定位多个物体

动画帧探测器

强力归整非整数帧

动画缩放/ 帧率转换

安装GraphEditor插件

 K帧工具：用来方便设key。

Timing工具：用来快速修改两帧间的时间间距，是针对时间滑条的功能。

中间张工具：用来在两帧间按一定的百分比加入中间BD张 / 极张。支持动画曲线或所选动画帧的单独处理。

切线类型工具：用来改变动画曲线点的tangent切线类型。类似曲线编辑器的功能，不过它是针对时间滑条的。

提示：每一栏的张合情况将自动记忆，下次开启帧工具箱将保持原先的张合状态。
