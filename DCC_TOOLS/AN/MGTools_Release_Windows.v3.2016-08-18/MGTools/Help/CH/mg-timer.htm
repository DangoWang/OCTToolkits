MG计时器按钮功能：

 


直接点击将打开MG计时计算器。

这个工具具有计时功能，及任何帧率转换及时间格式功能。

同时2.0版新加入计算器功能，方便计算帧数及安排工作进度!

 

MG计时计算器界面：

 

最上面为计时结果显示区 ：显示计时/帧率转换的结果。左上角的字母代表当前的输出单位。

提示：在这一区按右键，将出现各种帧率的marking Menu。将把当前结果以另一种帧率显示。

在这一区按左键，将出现记录/停止记录的marking Menu（如上图）。

第二行按钮区：

左边第一个按钮: 要转换帧率原始数据的单位。(在上面点右键选择其它单位)

输入框: 输入要转换帧率的原始数据。

->转换 !按钮 : 输出转换的目标数据。在上面点右键，可以选择输出单位。最后结果一样显示在上面结果显示区。

计时 -> (显示单位): 选择点击开始计时，括号内为将来显示出来的结果单位。点击后该按钮变成停止录制按钮，再按上就停止计时并显示结果 。

下部右方为结果记录区：

这个区将把每次转换/计时/计算的结果逐行记录下来。

如果是计时的，将显示计时结果：。。。

如果是转换帧率/显示方式的，将显示转换结果：。。。

如果是计算的，将显示计算的算式及结果：。。。

提示：在这一区按右键/清除历史记录： 将把记录清空。

         如果觉得这个一框的字体太小，可以按ctrl+鼠标滑轮来增大/缩小字体的显示。

计算器界面：

         算式输入框: 输入要计算的算式，如上图所示。注意，这个算式不管加减乘除，都是从左到右依次计算，所以无须担心运算符优先问题！

                 同时它也不支持任何括号，所以请只输入数字和".+-*/% "。注意%是用来做取余数运算的。

                 因为是可以自由输入，所以可以用鼠标激活这个输入框，用小键盘输入，按小键盘的enter键来得到结果。

         <-计时结果按钮: 将上方的计时/帧率转换结果输入到运算输入框的末尾。方便让计时的结果参与到下面的计算。

         其它->: 直接点击这个按钮，将弹出一个菜单。菜单里有很多其它运算，如乘方，求根，绝对值，倒数等等。

 清空按钮: 清空算式输入框。

 退格按钮: 将算式输入框的最右边一个字符删掉，相当于键盘的退格键。

 其它按钮: 相应按钮点击后将在算式输入框输入相应的数字/运算符。按=号按钮将产生计算结果，结果将留在输入框内，运算历史将出现在右边的历史栏。

         这个过程可以很方便地用小键盘上的数字/运算符按键来代劳。(当然要先把鼠标指针放在算式输入框)
