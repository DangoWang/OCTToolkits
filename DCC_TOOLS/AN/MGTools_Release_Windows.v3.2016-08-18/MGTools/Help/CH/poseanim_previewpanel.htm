Pose 动作库预览面板：

打开方式：点击主界面最右上方的按钮或在主界面内容区双击某一个项，就会在主界面右侧弹出这个预览面板。

在这里您可以浏览该Pose或动作的预览图，及预览视频。  

最上面一行，显示了当前预览的项的层级路径。
   内容是：  角色名称 > 类别名称 > 项名称: 预览图文件名称
 

在上方的预览图显示区，直接左键右击，可以弹出一些浏览用的marking menu.

    如果是预览动画项，在左下角还会有一个播放按钮。单击播放按钮播放当前视频，双击播放按钮播放该项所有预览视频。

 

中间的一行是按钮区，从左到右分别是：
   浏览前一项， 浏览后一项， 浏览上一张预览图， 浏览下一张预览图， 将当前预览图设为图标。
  如果向前向后的按钮不能用，则说明当前类别可能只有一个项，或该项只有一张预览图。

 

下方显示了该Pose/动画的一些信息，

   包含名称，作者，日期，时间范围，涉及到的Maya场景，Asset信息，

    还有修改历史。如果你编辑过该项，如重新指定了图标，在这里都会有记录。
