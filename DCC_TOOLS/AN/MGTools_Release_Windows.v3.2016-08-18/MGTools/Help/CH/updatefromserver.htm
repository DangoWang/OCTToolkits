从服务器端更新MGTools：

如果您的MGTools，是以服务器模式安装的，就可以使用这个功能。否则这个功能是灰掉不可用。

若已经以本地模式安装的MGTools,可以重新安装并选服务器安装模式，或者用MGTools Relocation工具。

访问方式：

▪在MGTools工具条的第一个图标点右键 更新/从服务器端更新MGTools..。

▪在MGTools菜单栏菜单 更新/从服务器端更新MGTools。

 
访问方式：

直接点更新，即可以将服务器上最新的MGToosl源更新到本地。

提示：这个更新动作，只会更新针对当前Maya版本的插件及数据，且不会动到MGTools_LocalData目录里的用户配置。
