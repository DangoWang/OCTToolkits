动画曲线切线类型工具功能：

 

用来改变动画曲线的切线类型，类似于Maya曲线编辑器里的一些切线类型按钮功能。

 

不过它是针对时间滑条的，意味着你可以在不打开曲线编辑器的情况下，快速改变帧的切线类型。

 

典型用法：

1.打开帧工具箱，点开切线类型工具界面如下图。

 

2.如果只是要自动平滑动画曲线，点修正Overshoot完事。

3.如果要改变切线类型，则按图中第二排按钮所示，选择任何一种切线类型。选中类型将加暗显示。

4.按最下面的两行按钮任一个来改变曲线的切线类型。

界面按钮的解释：        

头行的修正Overshoot按钮 ：将在自动把乱扭的，不顺的动画曲线修顺，处于曲线波峰/波谷的帧将自动打平，其它的以尽量不overshoot前提下调顺帧手柄角度。

第二行的动画曲线Tangent 切线类型按钮：跟曲线编辑器的图标类似，预选中其中一项类型即可。

左边： 改变当前帧的前一帧的切线类型; 如果当前帧有帧，则改变当前帧的切线类型。

两边： 改变当前帧的前后两帧的切线类型; 如果当前帧有帧，则改变当前帧及后一帧的切线类型。

右边： 改变当前帧的后一帧的切线类型。
全部： 改变所选物体所有动画曲线的切线类型。
所选时间范围： 改变所选物体所有动画曲线在所选时间范围内的切线类型。
