备份/还原动画功能：

     


点击将备份/还原当前选择物体的动画。这些动画信息存储于隐藏的locator中。

当你要做一个动画大改动前，可以备份下以便随时后悔。

 

备份/还原动画用法：

1.在场景中选择你想要备份其动画的层级，点击备份按钮即可。

2.会弹出提示对话框，提示备份的数目，被忽略的数目（因为其未被key帧）。

 
在场景里会生成一个locator，这个locator具有一切该层级的属性，包括自定义属性。同时帧被拷贝粘贴在上面。产生的层级如下图：

 

3.要还原动画，则再次选择该物体，点击还原动画按钮，如果之前有备份过动画，则动画被还原。
否则则警告：Warning: 您所的物体之前没备份过动画哦 :(

4.这个备份的动画信息将随场景保存。如果你想跨Maya重复利用，请用MG动作库的导出导入动画功能。
