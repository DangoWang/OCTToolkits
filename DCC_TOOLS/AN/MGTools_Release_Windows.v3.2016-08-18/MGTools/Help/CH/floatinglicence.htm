浮动授权 vs 固定授权

固定授权不可转移，跟机子绑定，不同机子需要不同授权， 一次性支付但需要为大的全新功能更新付费，但授权本身没有时间限制。

浮动授权可以在任何机子上登陆使用，并且不需要为更新付费，但需要按时间付费，并且需要连接网络，
即使网络不可用，最低限度也需要对两个特定的授权服务器IP开发访问， 可以通过ping来获得这两个需要支持访问的外网IP：
       ping ls1.mgland.com

       ping ls2.mgland.com

 

以浮动授权方式分发MGTools：

1.到http://mgland.com下载MG-LicenceConnector登陆器独立程序包。

2.这些程序是绿色软件， 可以直接解压到服务器。

3.到MGTools服务器上程序目录，找到MGTools/MGTools_ServerData/MG_LicenceConnector.txt这个文件。
用文本编辑软件打开，在 "ConnectorDirectory=" 后面，填入MG_LicenceConnector所在的实际目录。
可以写相对路径或绝对路径。

4.如果这个文件没有被填写，或者填写的路径对于动画师来讲不存在，则动画需要手动右键点击MGTools第一个图标，
选择“浮动授权/登陆。。”来手动拾取MG_LicenceConnector文件来设置其目录。

5.如果你机子的MGTools已注册了固定授权，则无论如何也会优先使用固定授权。
如果没有发现固定授权，而您已经设置好了MG_LicenceConnector目录，则会自动打开登陆器方便登陆。
提示：在Windows及MacOS, 你可以直接将登陆器关闭或最小化来隐藏它。
在Linux，因为所采用桌面环境很杂，Qt不能支持在所有些桌面环境上显示通知栏小图标，所以最小化时，它可能仍然在任务栏上占一个比较大的空间。
如果你的桌面环境可以正常显示通知栏小图标，则可以打开“Misc / Hide on minimize”， 这样最小化后就会隐藏。

6.如果你已经在登陆器里输入用户名和密码登陆， 但MGTools还是未注册的状态，则再次右键点击MGTools第一个图标，选择“浮动授权/登陆.."来更新状态。
 

避免每次手动输入用户名和密码：

在登陆器登陆界面，点上“Remember me”，这样下次就会自动登陆。

另一种方式，是将用户名和密码写入一个被加密过的文件。请查阅下面帮助。

 

如果您是系统管理员，并且用户名密码需要对用户保密：

1.使用MG-LicenceProtector密码保护器，这个程序随MG-LicenceConnector登陆器一同分布并被下载。

2.打开保护器，切到MGTools产品，输入用户名，密码。

3.在下面，你可以采用一些保护措施，保证您的授权不会被误用：
使用全局IP限制：输入公司采用的外网IP，这样这个授权只能通过这些IP来连接授权服务器。
使用用户码限制：采集MGTools用户码，这些用户码就代表着不同的机器。输入这些用户码，这只有这些机器可以连接授权服务器。
使用日期限制：如果您是自由职业者，只在公司短期时间，可以使用日期限制，这样过期后授权文件自然不能被使用。

4.点击Save按钮，将上面信息保存为.mgtoolslic文件。这个文件要被放置在MG-LicenceConnector登陆器旁边的"licences"文件夹里。

5.如果需要更新这些限制信息，每次编辑时，都需要重新输入密码，否则无法编辑。

6.如果授权文件被正确放置，则每次MG-LicenceConnector登陆器每次启动都会用这些文件自动登陆。
也可以通过登陆器上的"Login/out / Login All Server Licence(s).."来手动使用这些授权文件登陆。
