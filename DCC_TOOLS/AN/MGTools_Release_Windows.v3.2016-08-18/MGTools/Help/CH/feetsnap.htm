“定脚吸附”启动方法：

▪右击在MGTools工具条的图标即可。

▪直接点工具架上的图标(如果你有装全MGTools的Shelf)。

定脚方法：

当你用global层级移动来作位移而不是生key时，大部分情况就会需要定脚(除了直线式的走路)。具体方法如下：

1.选择你的脚步控制器，把时间指针移到要开始定脚的第一帧也就是脚刚落地的那一帧。以后每一帧都会定在这一状态。

2.点上述的图标，场景将在控制器轴心处产生一个暂用的Locator，并且时间向前跳一帧。

3.按键盘的“G“键，一直重复这一命令（也可以一直按图标)，同时观察视图上脚控制器的状态。

4.当动画到了脚必须离地的那一帧，停止按G键。

5.都吸附完了，可以在MGTools脚丫子图标上点右键往右划，清除所有暂用locator。如果这些locators有用来做约束或跟外部有连接，则不会被清掉。

  友情提醒：这个功能将逐帧设key在你的控制器上。

在定脚前，你必须保证脚本来就不会滑太多，要不然强制定脚会产生脚被硬拖住的现象。

也可以从脚离地前最后一帧倒退定脚，用的是向后定位吸附.

  提示：在Pro3.0版本中，选完要吸附的对象后，可以先在时间滑条上先选定一个时间范围，这样直接点一次吸附，则会吸附整个时间范围。

一般选择两个控制器的吸附方法：

向前/向后***吸附 是借用暂时自动产生的locator来标记位置作为参考来吸附的。

你也可以选择两个控制器作这个过程，这就和市面的吸附mel一个效果，它适用于Rig却在动画用地不多。
