从文件夹批量扫描添加资产项：

按资产库界面上的 按钮即可打开"从文件夹里批量读入资产项"的对话框:
 


 

扫描子目录：扫描拾取目录的子目录。

扫描深度：搜索文件夹的深度，只搜索当前文件夹意味着深度值为1.

同名则只用最新：如果在不同目录下搜到同样名称的maya档案，则只列出修改日期最新的档案。

MA/MB复选框：搜寻的maya档案是否包括mb/ma档。

拾取目录并扫描：点击将弹出拾取目录的对话框，拾取一个目录按确定后将开始扫描，然后把扫描结果放到此行下面的列表里。

 

每一行第一个勾勾上才会最终被导入资产管理器作为资产项。类别那一栏可以点击切换成其它类别。

：将当前扫描出来的所有资产项勾选上以便导入到资产管理器。

：将当前扫描出来的所有资产项去掉勾选。

：将当前扫描出来的所有资产项反选，之前勾选的变不勾选，不勾选的勾选上。

 "设置所选项为：设置按钮：将当前高亮的项设置为特定的类别。

：将当前扫描出来的勾选上的项导入到资产管理器当前工程内相对应的类别里。

    这个扫描操作往往会扫出很多的Maya档案项。请细心筛选它们，不必要或不常用的不要勾上。因为当前资产工程太多资产项的话，每次启动资产管理器会变地有点慢！
