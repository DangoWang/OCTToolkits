动画帧探测器功能：

在根据当前所选的对象，对其所在的最高层级底下的所有对象进行检查，

然后显示有Key动画的对象在下面的列表里。让您知道当前角色哪些物体被Key了动画。

有时角色明明key都杀掉了，但还是在动，这时可以用这个工具来查，到底哪个控制器的key还未杀干净:)

 

典型用法：

1.打开动画帧探测器界面如下图。

 

2.在场景里选择要探测的角色的任一部分，然后按"探测"按钮。

3.该角色上任何有Key帧的物体，将被显示在下面的列表里。可以点右键或直接双击列表项进行选择。

一些界面元素的解释：        

时间模式 ：

      全部帧：只要探测对象有被Key帧即显示在列表中。

      当前帧：只有探测对象在当帧有动画曲线经过才显示在列表中。

      包含帧的Infinity：有时虽然动画曲线没有经过当前帧，但是因为指定了帧的无限循环，如果勾上这个选项，则这种情况的物体也将被列出来。

只扫描transfofrm节点 ：若勾上将只探测当前Rig下所有的变形节点，如果不勾，则在该Rig总层级下的所有节点都将被探测(包含shape形节点等等)。

探测按钮：开始探测！

在场景中选择 ：在场景中选择列表中所选的项。

全部项目在场景中选择 ：在场景中选择列表中所有的项。
