MGTools按钮功能：

 

最大最小化MGTools. 第一次会在原位最小化，即缩成单按钮窗口，这时你可以定制它的最小化位置，大小，再点一下该按钮即还原。此时会记录下最小化信息以便下次最小化时自行判断用户希望的位置大小。

 
右键菜单功能：

     

MGLAND|Twincodes资讯.. ：打开MG-News程序，来看看有什么新工作或更新被发布;

MGTools 配置.. ：打开MGTools配置窗口，进行快捷键，自启动，shelf等相关设置；

记住当前风格：记住当前MGTools工具的长宽，屏幕位置。
风格选项：内置四种长宽风格：单列，双列，方形，单行。
                     自定义风格是不属于这几种的用户长宽配置。
                     点击任一项将快速设置MGTools工具条的长宽。
更新/从服务器端更新MGTools.. ：如果您的MGTools是以服务器端模式来安装的，
                    则可以用这个功能从服务器端获取MGTools新功能

更新/重新定位MGTools 程序.. ：打开MGTools Relocation工具，将MGtools安装模式转为服务器模式，或重新指定服务器端的MGTools源。

浮动授权/登陆.. ：打开MG-LicenceConnector登陆器来登陆浮动授权。

浮动授权/退出.. ：令MG-LicenceConnector登陆器退出当前授权，这将影响到所有在运行的本机maya, 同时密码将不会被记忆。

单机授权/注册MGTools.. ：打开MGTools注册窗口，这样您可以输入购买到的激活码来注册MGTools.
单机授权/向服务器端提交用户码: 如果您的MGTools是以服务器模式安装的，则可以用这个功能向服务器提交你机子的用户码，IP，用户名等信息。该信息存储在服务器端MGTools目录下的MGTools_LicenseData目录里。以便软件管理员批量注册。
单机授权/批量注册MGTools: 如果您的MGTools是以服务器模式安装的，则可以用这个功能批量注册已向服务器提交用户码信息的机子。请确保您是软件管理员。如果当前MGTools不是以服务器模式安装的，则会转到一般注册窗口。

单机授权/清除MGTools注册信息: 清除MGTools激活码等注册信息，注意这个功能不会清除试用期的日期信息。

安装/卸载/ MGTools 工具安装器.. ：打开MGTools工具架安装器，进行shelf安装操作；

安装/卸载/ MGTools 工具卸载器.. ：打开MGTools工具架卸载向导，进行彻底的MGTools卸载操作；

帮助..：查看MGTools帮助。

关于 MGTools..：查看MGTools版本历史及当前注册状态信息。

购买MGTools Pro授权..：访问MGTools网站，购买MGTools。

米古岛动画..：访问MGTools网站。

 

提示：点击按钮执行最小化将记录下正常窗口的长宽屏幕位置信息；

   再点一次按钮还原，将自动记录最小化的长宽，屏幕位置信息。
