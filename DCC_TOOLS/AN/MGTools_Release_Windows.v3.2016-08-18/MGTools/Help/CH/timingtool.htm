Timing工具功能：

 

用来改变当前帧前后两关键帧的时间间距。将移动当前帧后面的所有帧来改变这个间距。

 

典型用法：

1.打开帧工具箱，点开"Timeing工具"标题栏。界面如下图:

 

2.在场景里选择要改变时间间距的物体，并把当前帧放到想要改变间距的那一段时间内。

3.界面里上一栏是改变间距的绝对数值，下一栏是用相对增减方式来改变间距。

4.如果是绝对值方式，点改变按钮，如果是相对间距，按上图右下角的+>或-<按钮。

界面选项：        

设置前后帧的时间间距 ：用绝对值的方式，改变两关键帧间的绝对间距帧数。

      第一排数字按钮：快速在目标值基础上+/- 一定数值。也可以用下面的滑条或下下行的输入框来直接改变。

      获取按钮：获取当前帧两关键帧间的实际帧间距值。

      改变按钮：将当前帧两关键帧间的实际帧间距值改为输入框里所设定的值。                

加减前后帧的时间间距 ：用相对值的方式，增加/减少两关键帧间的相对间距帧数。

      用下面的滑条或输入框来改变即将加减的相对量，正数为增加，负数为减少。

      +>或<-按钮：将当前帧两关键帧间的实际帧间距值增加/减少左方输入框里所设定的值。
