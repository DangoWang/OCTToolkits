显示层集按钮功能：

   


打开显示层集配置界面，来管理显示层集。

这个工具主要为了使动画师提高建显示层的效率。

 

启动方法：

▪点击在工具架上点图标（如果你有装全MGTools工具shelf）打开显示层集管理界面；

▪在MGTools工具条上右击，在右键菜单选择 显示层集工具..
 

存在原因：

动画师开档的第一件事就是建显示层，每次都选物体建层并命名比较麻烦。你可以把这种层名和层成员的信息存成一种集，

积累到一定程度以后开档就点按钮调用这种集，它就可以自动侦察场景中是否有相应物体，有的话就建相应层并把相应物体

放进去，并且命名妥当。

 

显示层管理器界面：

左列表 ：显示当前已有的所有显示层集名称(也就是将来显示层名称)。

右列表：显示当前左列表所选显示层集的成员列表。

从当前场景选择新建集：根据场景所选物体建立显示层，这些物体将来就是这个显示层成员。(如果场景无选择，则按钮不可能)

从当前显示层新建集：根据Maya层编辑器所选显示层来建立显示层集。

加入当前场景选择：把当前场景所选物体加入当前左列表所选显示层集成为其成员。(如果场景无选择，则按钮不可能)

移除当前场景选择：把当前场景所选物体从当前左列表所选显示层集成员列表里排除掉。(如果场景无选择，则按钮不可能)

重命名 :重命名当前所选显示层集名称，也就是将来自动建立的显示层名称。

删除 :删除当前所选显示层集，也可以按键盘上的Del键。

从成员里移除：把当前右列表中所选物体从当前所选显示层集中排除掉，你也可以直接按键盘上的Del键排除掉。

使用全部集：把之前建立的显示层集都测试一遍，如果场景存在显示层集中成员物件，则自动建立相应显示层，

并把物件放于这一层。

刷新界面 : 刷新显示层集管理器的列表。

关闭 : 退出显示层集管理器。

  在建立了很多的显示层后，启动使用全部集检测该建哪些显示层并放入哪些物体时，难免遇到一个物件放入两个显示层的情况。

这就要求用户对自己所建的显示层里的成员有足够的了解和正确的命名，以免后面混乱。
