选择管理器按钮功能：

对场景中的选择做两次的暂存，

并可以进行选择的加减相交等等操作。

 

存在理由：

并不是动画师的任何选择操作都想要存成选择集，有时他只是需要一个地方暂时存放他的选择，以便后面加选其它对象。

这种临时的选择操作非常需要到。
有时候你想把相类似的角色动画互拷，用框选再拷贝，框选再粘贴动画的方式有时会乱掉，用选择管理器则可以一一对拷。

 

选择编辑器界面：

双列表区：一个是当前选择物体区，一个是上次选择物体区。两个列表下面的功能一样，作为运算的两个对象存在。在2.5版后，双击列表项则会在场景里选择相应的节点，并且在列表下面一栏显示出当前列表项的maya类型及其shape节点的maya类型(如果有)。

列表按钮区：

获取场景选择: 读取此时场景里选择的物体信息进相应左/右列表。旁边的>按钮点击有更多获取选择的方式。

重新选择列表: 重新选择列表里的项目。旁边的>按钮点击有更多将列表的控制器在场景里选择的方式。

重定向左/右列表: 把左/右列表的项目重定向并选择并反映在相应左/右列表。

拷贝列表动画: 把左/右列表的项目的动画拷贝到剪贴板。

粘贴列表动画: 粘贴之前拷贝的动画到左/右列表的项目。

备份列表选择: 把左/右列表的项目选择代码以locator方式存于场景。

恢复选择到列表..: 把先前备份于场景的选择信息重新使用并选择。

导出列表选择: 导出选择，实际将调用出新建选择集界面。

导入到列表: 导入选择，实际将调用出选择集界面。

清空列表 : 读取此时场景里选择的物体信息进左列表，清掉右列表。

关闭 : 关掉选择编辑器。

提示：在左/右列表上点右键亦可访问到相应功能，同时还有俩列表相加，相减相交的功能。部分功能在上面菜单栏也能访问到，只是跟列表无关而和当前场景实际选择有关。

在左/右列表里选择一些项，直接按键盘上的del键可以快速去掉所选项。

 

备份，恢复选择：

在场景里选择你想的物体后作下面任一操作可备份选择：

▪在选择管理器界面，把选择载入列表后，点备份列表选择 按钮。

▪在选择管理器菜单栏的Selection菜单里点备份列表选择菜单。

▪在选择集界面，在选择集项目点右键，将所选集随场景保存；

将弹出对话框，输入备份选择的名称。


 

点备份! 后，将在场景里建立这样的节点结构：

所以不用当心它破坏即有的场景干净度，那些选择用的代码将作为注释存在于这些locator中。

在恢复选择时只需要读取这些注释并执行。

要恢复选择，在选择管理器界面点恢复选择到列表按钮。

将弹出现在的场景备份列表：


 

双击列表上项目就可以重新选择之前备份的选择物体。这与恢复到场景效果一样。
恢复到左右列表！按钮则是把备份的选择列在选择管理器的左/右列表，同时在场景里选择相应物体。

备份/恢复选择，都基于随场景保存的locator节点，换一个场景将失去这些选择。如果你要跨场景使用，请用MG选择集功能。
