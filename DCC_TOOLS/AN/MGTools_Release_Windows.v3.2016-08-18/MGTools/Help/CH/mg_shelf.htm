MG工具架2.0按钮功能：

 

点击它将弹出MGShelf工具条，这个条显示的内容正是你之前内置的Mel.

在2.0版时基本重写了代码，使之支持Shelf的类别，及拖放操作。

 

右键菜单功能：
    这个右键菜单在此次MGtoosl运行期间第一次点的时候会稍慢，因为第一次要生成右键菜单，第二次点就很快！

"刷新该菜单"以上区域 ：内置的外部Mel将以菜单命令方式内置在这一区域的菜单中，并以类别作为母菜单组织起来；

刷新该菜单 ：刷新此右键菜单；

MGshelf 管理器 .. ：打开MGShelf管理器来内置/编辑外部mel；

Shelf 版本转换器 .. ：打开Shelf版本转换器来转换shelf文件的版本，使低版本maya可使用高版本的maya shelf ；

打开MG工具架文件夹 .. ：打开MGShelf文件实际存储的文件夹；

帮助..：启动MGShelf帮助。
