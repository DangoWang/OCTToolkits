如何新建库工程：

▪点击主界面左上方的按钮，即可以打开新建库工程对话框。

▪主界面库工程菜单/ 新建库工程..

▪如果当前本地没有任何库工程，直接点击层级显示栏上的新建库工程..按钮
 

新建库工程对话框：

在第一行输入库工程的名称，建议直接用你制作的动画项目的名称，不过注意，只能用字母数字和下划线。

因为将来需要实际建立一个该名称的目录。
 

库工程描述：输入一些描述文件，说明该工程状况，也可以不输入。

自定义全局根目录： 拾取一个服务器上，所有layout人员都可以访问到的路径。点重置按钮来清空，即使用本地库放弃全局库。

    注意，本地库的根目录是写死的，一定是在当前Maya用户目录/MGTools_GlobalData/MG_AssetLib/下面，并以各库工程名来组织。

资产项状态设置： 为您的资产库的资产，设定几个状态预置。

比如如果您是在家的soho，可以设立这些状态：Original_Rig, Net_Rig，代表是自已做的rig，还是网上下载的免费rig等。
一般 生产环境中，可以设立Proxy, Lo_Res, Hi_Res等，代表模型的高低模等。

也可能用working, pending, approved来表示Asset是否已通过。

▪左上的输入框： 如果左方的列表的所有状态都不是你要的，您需要在些输入自定义状态，并按"添加状态预置"按钮来加进去。
如果不想要状态，可以点垃圾桶，来删除列表中选中的状态。（工具自带默认就有的状态预置不能删除）

▪在列表中选中一些想添加到库工程中的状态，点击"添加所选状态预置到工程 ->"按钮，来将这个状态状态，添加到右边库工程状态的列表中。

▪选择库工程状态列表中的某一个或多个状态，点击中间颜色面板中某一颜色，点"指定颜色 ->" 按钮，将颜色应用到右边列表中选中的状态。

▪这样将来在主界面将某一资产项设置为该状态时，就会显示这个颜色。

确定按钮： 新建/确认编辑一个库工程。        

 

  提示：编辑即有库工程跟新建库工程菜共用同一个对话框，即上面所示的这个对话框。
