MG吸附工具按钮功能：

 

直接点击将向前PO吸附。（相当于ParentConstraint无offset效果）

这是MGTools被用到最多的功能，在吸附过程可以逐帧预视，直观，方便。

在2.5版本中，解决了一些重名的bug，并且在Maya 8.0及以后的版本优化了算法。

右键菜单功能：

 

 

清除吸咐标记器 ：清除Snap用的暂用Locators。如果这些locators有用来做约束或跟外部有连接，则不会被清掉。

标记器管理器.. ：打开Snap用locator管理器管理这些locators；

备份/还原动画..: 用Locator备份/还原所选控制器的动画。放在这里主要是有时逐帧吸附前备份动画。

剩余的都是吸附功能。下面将详细介绍。

很多吸附功能后面都有选项框，点击打开MG配置窗口，并转到吸附功能的配置页面：

   

     清除经过的非整数的位移旋转帧：如果勾选 ，则在向前或向后吸附或整个时间范围约束时，会将经过遇到非整数帧吸附成整数帧。

   当时间条上有选择范围时吸附整个范围：当时间滑条上有高亮时间范围时，自动吸附整个范围。

吸附方式有下面几种 :

定位吸附: 把第二选择物体吸附到第一选择位置，包括位移和旋转。原理同先做无offset的父子约束再删除约束。

向前定位吸附: 这是最最典型的用法。在POsnap基础上，每吸附一次向前进一帧。下面一章将介绍具体用法。

向后定位吸附 : 在定位吸附基础上，每吸附一次向后倒退一帧。

点吸附 : 把第二选择物体吸附到第一选择的点位置，只包括位移。原理同先做无offset的旋转约束再删除约束。

向前点吸附 : 在点吸附基础上，每吸附一次向前进一帧。

向后点吸附 : 在点吸附基础上，每吸附一次向后退一帧。

旋转吸附 : 把第二选择物体旋转和第一选择统一，只包括旋转。原理同先做无offset的旋转约束再删除约束。

向前旋转吸附 : 在旋转吸附基础上，每吸附一次向前进一帧。

向后旋转吸附 : 在旋转吸附基础上，每吸附一次向后退一帧。

向前相对吸附 : 在主要用来模拟offset的父子约束。每吸附一次向前进一帧。下下章将介绍具体用法。

向后相对吸附 : 主要用来模拟有offset的父子约束。每吸附一次向后退一帧。

向前相对点吸附 : 主要用来模拟有offset的点约束。每吸附一次向前进一帧。

向后相对点吸附: 主要用来模拟有offset的点约束。每吸附一次向后退一帧。

提示：在shelf架上如果有装全MGTools工具图标，将会有一些snap的图标，其中如果图标中有加号的，都是说明该吸附方式是每吸附一次后向前进一帧。
