建立角色：

▪点击界面左上角按钮/新建角色..

▪点击菜单栏角色菜单/新建角色..

▪如果当前库没有任何角色，直接点击层级显示栏上的新建角色..按钮  

▪右键点击界面左侧的角色列表区域/新建角色..

 

新建角色对话框：

在第一行输入角色的名称，请不要太特殊的符号，一定得用字母数字和下划线。因为将来需要实际建立一个该名称的目录。

添加角色类别：这一栏列表中选中的类别，将在角色创建后连带创建。

         因为角色下面还必须要有类别，所以这这个创建对话框中可以顺便创建类别。

自定义类别：可以输入一个自定义类别名称，并使用添加到预设按钮，让他添加到列表中。

         这样以后不管是添加角色还是添加类别对话框，都可以使用到这个自定义的类别预设。

创建按钮：创建角色及连带的类别并在界面上显示。

取消按钮：取消角色创建。

 

添加Pose动作库类别：

▪点击界面左上角按钮/新建类别..

▪点击菜单栏角色菜单/新建类别..

▪如果当前角色没有任何类别，直接点击层级显示栏上的新建类别..按钮  

▪右键点击界面左侧的角色列表区域/新建类别..

 

添加类别对话框：

上方列表中选中的类别将被创建。

注意，当前角色已经有的类别，不会再在列表中列出。所以有可能你会看到这个列表是空的，因为你把预置的都创建过了。

自定义类别：可以输入一个自定义类别名称，并使用添加到预设按钮，让他添加到列表中。

         这样以后不管是添加角色还是添加类别对话框，都可以使用到这个自定义的类别预设。

添加按钮：添加列表中选中的一个或多个类别并在界面上以tab的方式显示。

取消按钮：取消类别创建。
