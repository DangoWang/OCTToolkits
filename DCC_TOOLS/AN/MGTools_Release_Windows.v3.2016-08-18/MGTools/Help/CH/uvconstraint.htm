UV约束功能：

生成可以粘着于模型表面某一处的控制器，可以用这个控制器来约束别的物体。

有点类似于著名的Rivet脚本，不过它的好处是，即使模型光滑了也不会出错。

生成时支持的对象有： Polygon的点，线，面。

它要比顶点约束器的旋转要来得可靠，推荐使用。不过只有支持hair的Maya版本才能支持这功能。

 

用法：

 

1.选择Polygon的点，线，面这三种中的一种。

2.点击工具架的 按钮（如果有装全MGTools的shelf），或者约束工具条上的相应图标。

3.这时即生成一个U型的控制器，它属于一个nurbs curve物体。可以用它来约束其它物体，使该物体粘着于模型表面。
随表面变化（如blendshape等）而变化。

  生成的控制器的位移和旋转都是很稳定的。

  另外，生成的控制器的个数是由你选择的次级物体的个数来决定的。如果你选的是一个边，则生成一个控制器。如果你选的是两个顶点，就是每个顶点生成一个控制器。
