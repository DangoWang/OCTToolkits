欢迎使用MGTools Pro3+ !

 

MGTools是一个专门为maya角色动画师而制作的工具包，目前主要针对于动画流程及Layout流程，

主要是为了提高Maya动画操作过程的效率，从而使动画师可以更专注于角色的表演部分。

工具所有操作都是maya内部节点操作，不会影响到场景文件的通用性，因为不会添加任何自定义节点。

这个帮助文档将详细介绍每个工具的功能及使用方法。

MGTools 作者：

Miguel Winfield ( 高文峰 )，来自中国的3D角色动画师和技术开发人员， 目前在Animal Logic做研发。 欢迎点击 “关于米高” 链接来了解更多:)

关于MGTools 语言：

正式支持中英双语（中文包括简体及繁体中文），有些人习惯用英文，可以在MG配置窗口里设置切换。 :)

 
MGTools 帮助使用：

左侧的栏目主题与MGTools工具条的按钮图标一一对应，在工具界面中点帮助按钮也可以直接跳到相应帮助。
