路径描绘器按钮功能：

 

直接点击打开MG路径描绘器。

其功能主要是为了追踪显示物体的运动路径，并且可以以摄像机视角进行追踪。

 

 

启动方法：

▪点击MGTools工具条的按钮。

▪直接点工具架上的图标(如果你有装全MGTools的Shelf)。

 

界面：

 

 

: 点击来把当前场景所选物体添加到描绘列表。将生成该物体的运动曲线。描绘选项将以当前列表所选描绘项的选项为准。

: 将列表所选的描绘项从要描绘的列表里删掉，场景的动画路径也将被删除。也可以选列表项再按del键来做同样的事。

: 手动刷新当前列表所选描绘项的曲线。你也可以双击该列表项来手动刷新。

: 对摄像机的链接开关。后面显示了链接的摄像机名字。所谓的链接摄像机，就是当摄像机运动的情况下，描述出在该摄像机画面中所显示的
 物体运动路径。所以需要将之设置为“与该摄像机链接”。

"<" 按钮：选择一个摄像机，然后按该按钮将之与当前所选列表项链接。如果没选择摄像机，点击则将当前激活的视图摄像机自动加入。

: 刷新界面，以显示当前场景里已有的描绘项。

列表：

▪描绘对象：显示了要追踪描绘的节点对象名。双击这一项来手动刷新该项描绘结果。

▪自动：开关是否自动更新该项描绘。打开时，如果你有改变追踪对象的位移则会自动更新路径曲线。

▪路径：开关是否显示路径曲线。

▪帧：开关是否显示表示帧的圆球型曲线。

▪Key：开关是否显示表示关键帧的圆球型曲线。

 

选项栏 ：这一栏的标题显示了当前设置选项的描绘对象名字。

▪时间范围：显示了要追踪描绘的时间范围，点"<"按钮来将描绘时间范围设为当前Maya时间条显示的时间范围。

▪=Maya范围：按下后，每次更动Maya时间显示范围，则自动刷新该描绘结果。

▪路径 / 帧 / Key颜色：拖动后面滑条来实时改变相对应描绘组件的颜色。

▪大小：拖动后面滑条来实时放大 / 缩小帧/关键帧的大小。
