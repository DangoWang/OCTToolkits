>> 试用期: --------------------------------------------
未注册的MGTools将自动进入为期20天的试用期，试用期间除了不能在线更新以外，功能没有限制。
试用期间按一定天数会有对话框提醒，后最后几天会更频繁些，同时每次Maya启动会有HeadsUpMessage提醒余下的试用天数。

>> 试用版过期: -----------------------------
有很多功能会被锁掉，同时每天弹出一次提醒对话框。建议到http://mgland.com或http://twincodes.com购买商业授权。

>> 商业单机版授权: -------------------------------------
功能没有限制，也没有提醒，可免费进行小版本更新，对于大版本升级，如3.0到3.5， 3.5到4.0这种，需要另行支付一定的升级费用来升级授权，才能使用到这些升级带来的新功能。另外不同的计算机需要不同授权，不同类别的操作系统（Windows, MacOS, Linux）需要不同授权。同一机子，同一类型的操作系统，在不同Maya的版本上使用可以不用重新购买授权。

>> 商业浮动授权: -------------------------------------
功能没有限制，也没有提醒，授权有效期内永久免费更新，可以在任何机子上使用，同一时间能使用的授权受限于购买的授权个数。按一定的时间单位收费。
